# 美恐无限流创作工作流 Skill

本 skill 是《美恐最严厉的父亲》的执行卡，不替代正式设定文档。写作或审查前仍需读取：

- `CLAUDE.md`
- `《美恐最严厉的父亲》设定文档.md`
- `《美恐最严厉的父亲》剧情大纲与写法定稿.md`

若本 skill 与上述文档冲突，以上述正式文档为准。

## 开工前五问

每次设计副本、章节细纲或正文前，先回答：

1. 本章/副本的强识别物是什么？
2. 当前误导是什么：原作误导、类型误导、系统措辞误导，还是角色常识误导？
3. 规则代价在哪里出现：死亡、污染、身份替换、记忆缺口、救人付债，还是身体异常？
4. 林扬哪里误判、来不及、救不全或付债？
5. 父亲式反制落在哪个动作、物件、称呼、规则或现场站位上？

答不出前三项，不要直接写正文；答不出第四项，容易写成无代价爽文；答不出第五项，容易失去书名辨识度。

## 副本卡

设计副本时先填：

```text
副本类型外壳：
强识别物：
主场景：
入场初始信息：
原作/类型误导：
原创隐藏规则：
第一处异常：
第一次死亡或不可逆污染：
林扬第一次误判：
林扬第一次小型反制：
关键配角/路人代价：
父亲式压场场面：
通关方式：
可见收益：
长期债务/污染线索：
后续回收点：
```

前两卷的主菜必须有可识别欧美恐怖外壳，原创内容优先作为隐藏规则、误导机制、短间章或系统污染线索。

## 章节卡

设计单章时先填：

```text
章节标题：
上章钩子：
本章主场景：
强识别物/关键物件：
本章恐怖压迫：
本章规则线索：
本章试错代价：
林扬判断：
林扬误判或代价：
林扬反制动作：
配角恐惧/分歧：
本章可见收益：
章末钩子：
```

每章至少承担两项功能：建立恐怖压迫、推进规则探索、制造试错代价、强化林扬魅力、推进配角关系、展示父亲式反制、增加可见收益、留下章末钩子。

同时必须控制章节容量：每章只围绕一个核心问题和一个主钩子展开。不要把入场/转场、凶宅群像、关键误导识破、路人污染、系统弹窗、新钥匙/新地图和核心规则揭示全部塞进同一章。强钩子出现后要给读者发酵空间，代价、系统确认、规则反转和新地图入口要分章接力。

卷细纲与章节正文应按功能接力设计：

- 钩子章/场景开启章：强异常、强识别物或主角关键行动，抛出下一问题。
- 展开章/探索章：展开场景、人物阻力和规则线索，只解决一部分问题。
- 代价章/污染章：用死亡、污染、替换、记忆缺口或付债验证规则重量。
- 反制章/关系章：林扬基于现场信息完成小型反制、救人、压场或关系推进。
- 揭示章/转折章：揭示当前层答案，同时制造下一层更具体的问题。

## 正文节奏

默认执行：

- 70% 严肃压迫。
- 20% 冷静拆解。
- 10% 黑色幽默。

顺序优先：

1. 先写环境和日常物件的不对劲。
2. 再写配角正常恐惧、误解或自我安慰。
3. 让试探产生伤害、失踪、污染或身份异常。
4. 林扬用一句话切开恐惧，但不能把死亡写轻。
5. 怪物或规则回应他的挑衅。
6. 林扬根据现场信息反制，不靠全知攻略。

## 林扬口吻

林扬可以荒诞、嘴欠、冷静分类，但每句怪话必须服务至少一项：

- 观察异常。
- 缩小规则范围。
- 试探怪物或系统。
- 稳住配角。
- 压制恶人。
- 触发父亲式反制。

禁止把他写成游戏主播、纯吐槽役、冷漠实验员或无代价救世主。

## 代价规则

副本探索必须让规则有重量。可用代价包括：

- 死亡。
- 不可逆污染。
- 身份替换。
- 记忆缺口。
- 亲人误认。
- 身体局部异常。
- 救人付债。
- 被规则标记。

无辜者可以死亡，但频率要低、后果要重、信息要关键。林扬不能主动牺牲无辜者刷情报，但可以因为信息不足、时间不足、能力不足而救不全。

## 审查清单

审查副本、章节细纲或正文时逐项检查：

- 是否有可识别欧美恐怖外壳和强识别物。
- 林扬入场是否不知道片名、怪物真实身份和完整规则。
- 原作或类型知识是否只是半真半假的低级情报。
- 是否有原创隐藏规则，而不是照搬原作流程。
- 恐怖是否先成立，黑色幽默是否没有冲淡死亡重量。
- 死亡、污染、替换或记忆缺口是否留下有效信息。
- 林扬是否有误判、救不全、付债或承担后果。
- 父亲式严厉是否先有承担、救人、立规矩或付代价支撑。
- 系统是否没有用万能黑箱抹掉现场推理。
- 章末是否有新异常、新代价、新规则误判、新系统措辞变化或新债务。
- 本章是否容量过载：是否同时塞进入场、场景展开、规则揭示、代价落地、系统解释和新地图入口；如果过载，优先拆章。
- 本章主钩子是否清晰唯一，是否被后续更大的钩子覆盖。

## 常用任务提示

副本设计：

```text
使用 webnovel-workflow 与 meikong-horror-workflow，设计一个《美恐最严厉的父亲》副本。先填副本卡，再给卷纲/章节推进。重点写强识别物、原作/类型误导、原创隐藏规则、第一次代价、林扬误判和父亲式反制。
```

章节细纲：

```text
使用 meikong-horror-workflow，为第XXX章写章节细纲。先回答开工前五问，再填章节卡。不得让林扬全知，不得用系统面板替代现场探索。
```

正文审查：

```text
使用 meikong-horror-workflow 只读审查第XXX章，重点检查恐怖是否成立、死亡是否有重量、林扬是否只嘴欠不承担、规则探索是否像说明书、章末钩子是否有效。
```

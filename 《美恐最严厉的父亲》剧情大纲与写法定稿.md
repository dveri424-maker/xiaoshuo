# 《美恐最严厉的父亲》剧情大纲与写法定稿

本文档为《美恐最严厉的父亲》的剧情结构与正文写法执行稿。主设定文档负责世界观和能力体系，`CLAUDE.md` 负责项目执行规则，本文档负责后续写卷纲、章节细纲、正文时的剧情和文风共识。

## 一、最终定稿方向

本书采用：

> 可识别欧美恐怖模板 + 原创隐藏规则 + 入场未知识别 + 试错代价 + 父亲式反制 + 副本收益/债务。

“美恐”定位不是纯粹原作巡礼，也不是纯原创规则怪谈，而是以欧美恐怖影视和都市传说气质为外壳的无限流。读者要能快速识别凶宅、驱魔、鬼娃、杀人狂、邪教、录像诅咒、死亡规则、精神病院、怪诞旅馆等类型味道；真正的未知恐惧来自系统添加的原创隐藏规则。

主角林扬进入副本时不知道片名，不知道怪物真实身份，也没有原作名单。原作记忆必须在现场线索足够明确后才发挥作用，而且只能作为半真半假的低级情报。

## 二、第一卷方向

第一卷采用：

> 凶宅家庭诅咒 + 驱魔误导 + 原创隐藏家规。

### 商业理由

- 凶宅、驱魔、家庭诅咒是欧美恐怖最容易识别的入口。
- 老宅、房契、阁楼脚步、儿童涂鸦、家庭合照、地下室、神父、驱魔仪式等物件能快速建立读者期待。
- “家庭秩序崩坏”天然贴合书名里的“父亲”主题。
- 驱魔故事容易设置误导：读者以为目标是驱鬼，真正规则却是房子在挑选新的父亲。

### 第一卷核心规则建议

表层规则：

- 房子闹鬼。
- 孩子看见不存在的人。
- 母亲隐瞒旧案。
- 神父或灵媒认为这是附身或恶灵占屋。

隐藏规则：

- 家庭成员越互相隐瞒，房子越能改写他们的身份。
- 房子不是单纯杀人，而是在补全一个“家庭结构”。
- 驱魔仪式不是救人，而是可能给房子登记新的监护人。
- 被房子承认为“父亲”的人，会获得保护权，也会被房子索取责任。

林扬误判路线：

1. 起初判断为普通凶宅或附身事件。
2. 发现驱魔仪式没有削弱恶灵，反而让房间布局更像“家庭档案”。
3. 发现死者不是随机受害者，而是被房子判定为“不合格家庭成员”。
4. 最终确认副本核心不是驱鬼，而是争夺“父亲/监护人”的定义权。

### 开篇硬节奏

1. 第一章：必须出现强欧美恐怖识别物，例如房契、儿童涂鸦、驱魔神父、阁楼脚步、家庭合照错位、地下室门自己打开。
2. 第二章：必须出现规则死亡或不可逆污染。
3. 第三章：林扬给出一个错误或半真判断，并完成第一次小型反制或救人。
4. 五章内：揭示原作知识不可靠，甚至会害死人。
5. 第一卷中段：揭开原创隐藏家规。

## 三、正文写法比例

正文执行比例：

- 70% 严肃压迫。
- 20% 冷静拆解。
- 10% 黑色幽默。

恐怖机制主要参考《神秘复苏》：

- 规则型灵异。
- 媒介污染。
- 空间错位。
- 试错代价。
- 死亡和污染推动推理。

主角口吻少量吸收《玩家凶猛》：

- 冷静分类。
- 任务拆解。
- 反常吐槽。
- 副本多样性。

但最终读感必须服务本书自身辨识度：

- 严厉父亲。
- 规则整改。
- 荒诞理性。
- 恐怖反压。

## 四、黑色幽默边界

黑色幽默只能用于：

- 观察。
- 推理。
- 压场。
- 反制。
- 建立林扬辨识度。

禁止：

- 死亡现场廉价玩梗。
- 恐怖未成立前急着吐槽。
- 把林扬写成游戏主播。
- 把林扬写成冷漠实验员。
- 把怪物长期小丑化。
- 用段子抹掉配角恐惧和死亡重量。

正确节奏：

1. 先写压迫。
2. 再写配角恐慌。
3. 然后出现代价。
4. 林扬开口切开恐惧。
5. 怪物或规则立刻回应。
6. 林扬开始反制。

## 五、牺牲与污染

铁律：

> 林扬能赢，但不能全保。

牺牲不是为了清人数，而是为了验证规则、校准危险、制造主角压力。

可用代价：

- 死亡。
- 不可逆污染。
- 身份替换。
- 记忆缺口。
- 亲人误认。
- 身体局部异常。
- 救人付债。
- 被规则标记。

使用原则：

- 无辜者可以死亡，但频率要低、后果要重、信息要关键。
- 恶人、作死者、主动害人者可以优先承担高危试错。
- 重要配角可以被污染、替换、失忆、欠债，不必每次死亡。
- 每个副本至少安排一次林扬救下本该死亡或本该被规则吞掉的人。
- 每个副本也至少要有一次林扬救不全、判断失误、付出代价或承担债务。

## 六、父亲式严厉底线

林扬的“父亲式严厉”不是爹味说教，也不是虐待型控制。

他可以：

- 嘴欠。
- 冷静。
- 训斥。
- 压迫恶人。
- 拿规则反制怪物。
- 把崩溃的人从错误行为里拽出来。

他不能：

- 主动牺牲无辜者刷情报。
- 把受害者当工具。
- 在孩子和弱者受害时冷眼旁观。
- 用漂亮话包装自己的错误。
- 一直训人却不承担代价。

读者必须持续看到：

- 他能救人。
- 他能立规矩。
- 他能承担后果。
- 他能让恐怖反过来害怕他。

## 七、系统主线

系统不是中立任务栏，而是纵容、交易、改写恐惧规则的问题源。

系统的恶意表现：

- 用原作词汇包装误导任务。
- 隐瞒关键触发条件。
- 把奖励和债务绑定。
- 把通关副本的残响拼接到后续世界。
- 把被救幸存者、被训服的鬼、被改写的房子变成后续主线锚点。
- 让林扬的家规在其他世界留下污染痕迹。
- 把失败前任玩家的遗物、错误笔记、残缺尸体放进副本。

系统的边界：

- 不能无因无据随意改规则。
- 不能没有线索地强行反转。
- 不能用万能黑箱抹掉林扬推理成果。
- 不能让任务面板替代现场探索。

系统恶意必须有可见痕迹：

- 任务措辞变动。
- 奖励带副作用。
- 门牌号改变。
- 旧档案出现林扬名字。
- 幸存者记忆错位。
- 怪物提前学会林扬的口吻。

## 八、副本结构模板

每个副本设计时先填：

```text
副本类型外壳：
强识别物：
主场景：
原作/类型误导：
原创核心规则：
第一处异常：
第一次死亡或不可逆污染：
林扬第一次误判：
林扬第一次小型反制：
关键配角/路人代价：
父亲式压场场面：
通关方式：
可见收益：
长期债务/污染线索：
后续回收点：
```

每个副本必须具备：

- 可识别欧美恐怖外壳。
- 原创核心规则。
- 原作误导点。
- 至少一次死亡或不可逆污染验证。
- 至少一次林扬失败、误判、付债或救不全。
- 至少一次父亲式压场。
- 至少一种长期收益。

## 九、每卷收益链

每卷至少沉淀一种长期收益，避免单元剧通关后归零。

收益类型：

- 道具：错题本、戒尺、旧录音机、裂纹镜片等。
- 权柄：训诫权、监护权、点名权、旁听权、家规权等。
- 债务：被救者欠人情，被训服的鬼欠回应，被放过的怪物欠不杀之约。
- 关系：幸存者、灵媒、警探、神父、医生、被污染者。
- 漏洞：系统任务措辞、门牌号、错误副本入口、残响拼接规律。
- 污染线索：林扬的家规在其他世界留下痕迹。

收益必须可见，最好通过物件、称呼、门牌、档案、债条、钥匙、录音、照片、规则批注呈现。

## 十、禁止写法

- 纯原创慢热开局。
- 第一卷看不出欧美恐怖类型。
- 原作角色只当彩蛋或工具人。
- 游戏主播腔。
- 纯吐槽流。
- 死亡现场玩梗。
- 恐怖未成立前先抖包袱。
- 规则说明书式正文。
- 无代价救全员。
- 系统万能黑箱。
- 林扬只旁观总结，不承担危险。
- 父亲式严厉写成爹味说教。

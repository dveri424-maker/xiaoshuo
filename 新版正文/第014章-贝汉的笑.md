# 第十四章 贝汉的笑

约翰尼·贝汉进门前，先把鞋底在门槛外蹭了两下。

那动作体面，也讨厌。他把治安官办公室地板上的血、灯油和黑火药灰，全留在干净鞋底之外；鞋面不脏，昨夜那颗冲怀特胸口来的子弹就能先和县里隔开一层。

伊莱亚斯站在门槛内，没有让开。胸前那枚临时副治安官徽章没有遮，银面旧，却压在黑色外套上，贴得他胸口有点凉。桌上摆着扣押清单、乔伊口供、卷毛比尔的枪械登记纸、裂开的徽章盒盖。证物箱在汤姆手边，铜扣合着，谁想碰都得先过伊莱亚斯的眼。

贝汉的视线先落在徽章上，又落到证物箱，最后才回到怀特脸上。

“弗雷德。”他摘下帽子，笑得亲热，“听说昨夜出了乱子。我来看看你。”

怀特靠在椅背上，脸色难看：“看我，还是看犯人？”

“顺路都看。”贝汉把手套摘下，慢慢放到桌边，“卷毛比尔不是普通醉鬼。他牵着牛仔帮、边境旧案和县里的麻烦。你受了伤，我不想让墓碑镇因为一时火气，把事情弄得更难收拾。”

克拉拉站在窗边，笔尖已经落纸。

贝汉看向她，笑意不减：“报馆小姐，这句可以写。‘更难收拾’比‘街头枪声’文雅些。”

克拉拉说：“我写谁想收走枪。”

“那你今天有得忙。”

贝汉从内袋取出一份文书。纸折得整齐，封口处有红蜡，蜡上压着县印。印边有点糊，红得很新，新得像刚从哪间暖屋里端出来，还没沾过墓碑镇一粒灰。

他把文书推给怀特，却避开了伊莱亚斯那张扣押清单。

伊莱亚斯看见他的手指绕开清单边角，也看见他带来的车夫站在门外，靴子擦得很亮，可马车轮辋上还沾着南街红泥。那泥昨夜只在碎街灯旁被踩得最烂。县里人嘴上来得晚，车轮已经认得路。

“科奇斯县权力正在整理。”贝汉说，“墓碑镇不能把一个会牵出边境牛仔帮的犯人关在木板牢房里。卷毛比尔、涉案枪械、证人口供，移交县里保管。你养伤，韦德先生交出临时扣押物。对谁都体面。”

他身后的助手把一本窄窄的收据簿打开，递到怀特桌边。簿子上已经画好三栏：犯人、枪械、证人。最下方空着两处签名，一处给墓碑镇，一处给县里。那助手连笔帽都拔好了，笔尖悬在空栏上，神情谨慎得很，仿佛只要怀特点一下头，卷毛比尔就会自己从牢里滚进县里的马车。

这人连抢字都没说，先把笔和空栏摆出来，逼人自己把东西送进他手里。

汤姆忍不住：“昨夜你们县里的人在哪儿？”

贝汉没看他：“年轻助手，县里文书不按昨夜谁站在哪儿算分量。”

“那按谁鞋底干净算？”汤姆嘀咕。

门外爆出一声笑，又赶紧压住。

贝汉笑容没变，眼神却冷了一点。

伊莱亚斯伸手，把文书拿到自己面前。

贝汉终于正眼看他。

“韦德先生，你枪法很好。墓碑镇需要这样的枪手。但枪手不该把自己误认成法院。”

门外有人低声道：“昨夜法院可没打掉卷毛比尔的手。”

贝汉没有回头。

伊莱亚斯打开文书，没有立刻读。他先看纸角，再看墨迹，最后看封蜡。指尖碰到红蜡时，纸页边缘发冷，蜡印阴影里压着一点暗色，很快又被灯光盖住。墙上裂开的徽章盒轻轻响了一下。

贝汉的笑还在，右手拇指却摩挲了两下手套边。

这点动作很轻，轻到门外的人不会注意。伊莱亚斯却看得清楚。他没看贝汉的笑，只看那只手套又在杯沿上擦了一下。

“要读多久？”贝汉问。

“读到它说完谎。”伊莱亚斯说。

克拉拉的笔停了一瞬，又更快地写。

贝汉轻叹：“年轻人总喜欢把不懂的手续叫谎。”

伊莱亚斯把文书摊平，没有急着摆出审案架势，反而先把它垫在裂开的徽章盒盖上。盒盖不平，红蜡印便跟着晃了晃，像一个体面人忽然踩进了泥坑，还硬要说自己鞋底干净。

贝汉的笑淡了一点。

伊莱亚斯指尖停在日期上：“这份移交令写在昨夜街灯碎裂前一个钟头。”

贝汉道：“县里处理牛仔帮，不是从昨夜才开始。”

“那时卷毛比尔还没被扣，枪还没进箱，乔伊还没按手印。”伊莱亚斯抬眼，“县里提前替我们想好了要交什么？”

贝汉没有接这句，只朝助手偏了偏下巴。助手会意，忙把收据簿又往前推半寸，想用签名栏把话题盖过去。

伊莱亚斯伸手，把那本簿子按住。

“别忙。”他说，“你这上面也写了枪械。”

助手低头一看，脸色先白了一点。贝汉的眼神扫过去，他又硬把脖子挺直。

文书物品栏里写着“卷毛比尔使用短枪一把”。

汤姆打开证物箱。长管左轮躺在里面，枪口黑痕还没擦。门外的人伸脖子看，后排没看清，前排矿工却看得真切，骂了句：“短你奶奶，那枪管能戳进你鼻孔。”

怀特差点咳笑，胸口疼得皱眉。克拉拉低头咬了下唇，笔尖没停。

贝汉嘴角低了一点，又抬起来：“书记员笔误。”

伊莱亚斯拿起那把长管左轮，放到文书的“短枪”两字旁。枪管一压，纸面立刻翘起一角，那两个字被压得可怜巴巴，倒比刚才顺眼。

“那就请书记员来，把鼻孔量一量。”

门外笑声这次大了些。连那个刚才还替县里说好话的酒馆闲汉，也把帽檐压低，肩膀一抖一抖。

贝汉不笑了半息。

伊莱亚斯没有让他把那半息遮回去。他把左轮推回证物箱，又点向证人栏。

“三名证人。”他说，“昨夜现场登记两个，乔伊清晨补上。第三个是谁？”

贝汉沉默半息。

“县里有线人。”

“名字。”

“县里线人受保护。”

“那就让他受县里保护。”伊莱亚斯把乔伊的口供纸往自己这边拉了半寸，“墓碑镇的证人，不跟一个没名字的人同车。”

贝汉把手套重新拿回掌心，慢慢抚平指节位置。那双手套柔软、干净，和桌上这些火药灰、红手印、破盒盖摆在一起，体面得近乎可疑。

“韦德先生，你在用枪手口吻审县里文书。”

伊莱亚斯把文书往前推半寸。

“我在用你的文书审你。”

门外的杂音低了一截。贝汉听见了，笑意变浅。

他转向怀特：“弗雷德，你确定让一个临时副手这样说话？县里可以尊重你受伤，也可以重新考虑墓碑镇治安官办公室今后的地位。”

怀特脸色难看，却没低头。

“我签了临时授权。”

“临时授权不能对抗县权。”

“写错的县权也不能带走犯人。”克拉拉忽然说。

贝汉看过去，眼神冷了一点：“小姐，你最好知道自己在写什么。”

克拉拉把报纸样张翻到空白处：“我知道。‘贝汉先生拒绝解释文书日期’。这句不长，省铅字。”

贝汉盯着她片刻，笑出了声。

“墓碑镇真热闹。旅店开后门，报馆审文书，外来枪手扣犯人。弗雷德，你养伤这一天，办公室换主人了？”

这话说得漂亮，也毒。门外有两个人立刻去看怀特，像是等老治安官被这一句话扎出血来。

怀特的脸确实更难看了，却不是被扎疼，是被人隔着伤口摸了一把。

伊莱亚斯解下胸前临时徽章，放到桌上。

银面压在扣押清单顶端，闷响一声。那一下并不重，门外却有几个人下意识看过来。

“徽章不够。”他说，“得有人守住犯人、枪和证人，不让它们半夜不见。”

贝汉看着那枚徽章，没有接话。

伊莱亚斯继续翻文书。

“第四处。口供栏写证人自愿随县里转移。乔伊不会写字，今天才按手印。昨夜谁替他自愿？”

乔伊的口供纸摆在旁边。红手印还新，露丝的黄铜钥匙压着纸角。贝汉看见那把钥匙，笑容终于有一点僵。

“你把一个孩子推到县权对面，不觉得危险？”

“危险来自拿五美元买手印的人。”

“你有证据？”

伊莱亚斯点了点扣押清单：“抢人者有手印。传话人有签名。乔伊有证词。你的文书有日期。”

贝汉低头看桌上的东西。

纸、枪、钥匙、手印、裂盒盖摊在桌上，边角全沾着泥灰和旧血。贝汉的手从纸边掠过去，又停在半空，竟没找到一件能单独抽走的。贝汉带来的助手原本笔尖悬在签名线上，这会儿手腕越来越僵，墨水终于滴了一点在空栏边，正好落在“犯人”两个字旁。

门外的人先看文书，再看收据簿，最后看贝汉的马车。车夫坐直了些，没再伸手去拉车门。

牢房里，卷毛比尔忽然踹了一脚铁门。

“贝汉！你他妈到底带不带我走？”

屋里静了一下。

贝汉脸上的笑停了一下。

伊莱亚斯看向牢房，语气很平：“他也想知道谁说了算。”

卷毛比尔张嘴想骂，伊莱亚斯把长管左轮往证物箱边一放。枪身轻响，牢房里的骂声立刻低了半截。

贝汉重新笑起来。

“韦德先生，我承认，你很会摆证据。”

“我还会收证据。”

贝汉眼角一动。

伊莱亚斯把文书折回原样，没有还给他，而是放进证物箱旁边的空格。

“这份文书暂扣。理由：时间、枪型、证人数量、自愿转移栏存在疑点。贝汉先生可以留下副本，也可以让图森回电解释。”

贝汉伸手按住文书边缘。

汤姆立刻摸枪。门外几个镇民往后退，却没有散。怀特的手也压到桌面，眼睛盯着贝汉的手套。

伊莱亚斯伸手，压在文书另一边。他没有戴手套，手背上还有昨夜留下的浅伤。贝汉的手隔着软皮，他的手直接压着纸。

“拿走它，”伊莱亚斯说，“就在清单上签名。”

贝汉第一次不笑了。

两人的手隔着文书僵持片刻。街外马槽边，马打了个响鼻。卷毛比尔又踹铁门，铁门响得刺耳，却没人回头。

最后，贝汉松开手。

“你会后悔让县里难堪。”

“先让县里把字写对。”

贝汉把收据簿合上，动作不重。那声合页响却让门口几个人抬了抬眉。空签名栏还亮在桌边，文书也已经进了证物箱，贝汉只合上了一本没派上用场的簿子。助手慌忙去擦那滴墨，越擦越像个小黑眼睛，瞪着他自己。

贝汉把手套戴回去，转身前看了克拉拉一眼。

“报馆小姐，写的时候小心。墨水会留下。”

克拉拉抬笔：“我正要写墨水。”

贝汉离开时，门外的人让路比他进来时更快，却少了那点慌。他的助手上车时踩空半步，险些把收据簿摔进尘土里，幸亏车夫眼疾手快扶了一把，只是那扶人的手刚伸出去，门口便有人嘀咕：“小心，别把县权也扶歪了。”

他的马车碾过碎玻璃，响声拖过半条街。

怀特长长吐气：“你把他得罪死了。”

“还不够。”

伊莱亚斯拿起那份文书，重新摊到灯下。

克拉拉凑近：“还有哪里不对？”

伊莱亚斯没有看正文。他盯着红蜡上的县印，指尖沿着圆边停在左下角。那里缺了一小块，边缘整齐，压出来的痕比别处浅。

指腹贴着那道缺口，冷意从红蜡下透出来，隔着纸轻轻咬了他一下。

他把手按住，没有让自己往玄乎处想。

“这枚县印，”伊莱亚斯说，“少了一口。”

# 第十六章 白天的徽章

伊莱亚斯把图森回电钉在治安官办公室门框上。

纸还带着电报局的潮气，钉子敲进木头，台阶下的吵声被敲断了一截。矿工、报童、酒馆闲汉全抬头看。莉迪亚补在纸下的一行铅笔字很短：未见备案。

弗雷德·怀特坐在桌后，右肩缠着白布。昨夜那颗本该要他命的子弹没进胸口，只打碎了徽章盒，木屑刮进肩肉。林月兰刚换过药，药味压不住枪油味。

伊莱亚斯把五样东西推到日光里。

卷毛比尔的左轮，工资箱锁片，证人口供纸，缺口县印文书副本，还有他胸前那枚临时副治安官徽章。

徽章白天不亮，旧磨痕和弹屑刮出的细伤反倒更清楚。

卷毛比尔·布罗修斯被扣押绳绑在椅子上，右腕缠着纱布，边缘透出血色，嘴角有血。他抬起下巴，扫过门外矿工，还想把那点恶名挂回脸上。

“你们真以为一张电报纸能关住我？”他啐出血沫，“怀特老了。外来的小白脸戴半天破徽章，就敢管牛仔帮？”

门外几个矿工脚尖往后挪。卷毛比尔的名声在边境跑得快，酒馆里说起他，常有人先摸枪套。报童把号外卷到胸前，眼睛却还盯着伊莱亚斯。

伊莱亚斯没有拔枪。他拿起卷毛比尔的左轮，打开枪膛。

五发完整，一格空着。

“昨夜你说枪走火。”伊莱亚斯把枪膛转给门外看，“空膛擦过，枪口也擦过。撞针边还留着火药灰，替你擦枪的人手不细。”

卷毛比尔扯嘴：“枪匠会擦枪。你拿这个吓谁？”

伊莱亚斯把工资箱锁片推过去。锁片碰到枪管，发出短响。那一瞬，枪膛深处有一点冷意贴住他的指腹，空膛里传出一记极轻的马刺声。

他没有停顿。

“霍布炉灰里找出的锁片，锉痕和这把枪的击锤锉痕相同。”伊莱亚斯说，“同一把细锉，昨夜进过枪匠铺，也进过东方酒馆后桌。”

克拉拉的笔尖立刻动起来。

卷毛比尔盯着她：“报纸姑娘，你写错一个字，半夜就有人敲你窗。”

克拉拉没抬头：“那我把窗开着。省得你们又撬门。”

门外有人笑了一声，马上憋回去。笑憋得太急，有个矿工呛了口唾沫，咳得满脸煤灰。

这一声笑很要命。卷毛比尔靠恶名吃饭，别人怕他，他才值钱；别人一笑，他那身皮就先破了个口子。报童把号外摊开，拿铅笔在边角补了两个字：白天。旁边卖烟草的老头探头看了一眼，立刻把柜台上写着赊账的木牌翻过去，嘴里嘟囔：“夜里吓人，白天也得还钱。”

卷毛比尔听见这句，脖子上的青筋跳起来。他最想骂那个烟草老头，可伊莱亚斯的手还压着枪，骂人的力气没地方落，只能从鼻子里喷粗气。

贝汉站在办公室右侧，衣服整齐，袖口没有尘。他身边的助手捧着皮文书包，脸色比昨夜白。

“韦德先生。”贝汉开口，“回电未备案，只能说明文书传递出了差错。卷毛比尔必须交给县里。枪械、口供和工资箱线索，也要随犯人移交。”

怀特想起身，牵动伤口，眉头一皱。伊莱亚斯抬手，让他坐着。

“先看印。”伊莱亚斯把接管文书副本推到贝汉面前。

贝汉没伸手：“这只是副本。”

“副本来自你昨天留下的接管文书。”伊莱亚斯指向印章下沿，“缺口在左下。图森说没备案，你的人却带着它要犯人、要证物、要工资箱线索。”

贝汉助手喉结动了一下。

伊莱亚斯转头看他：“你昨天说，这枚章在图森盖的。”

助手把文书包抱紧：“我只是递文书。”

“谁给你的？”

“贝汉先生的办公室。”

贝汉手杖敲了下地板：“问话要有界限。”

伊莱亚斯把证人口供纸放到卷毛比尔面前：“界限在这里。灰毡帽供出换马牌，假报工供出火油，枪匠交出锁片。三个人都提到一个卷头发、带酒气、坐在东方酒馆后桌的人。”

卷毛比尔扭头：“满街都是喝酒的卷头发。”

伊莱亚斯把左轮推过去，枪柄朝他：“那就按手印，说这枪昨夜不在你手里。”

卷毛比尔脸上的笑收了。

门外矿工又往前挤。艾达也在其中，身上矿灰未洗，手里拎着工棚钥匙。她盯着那把枪，嗓音压着火。

“按啊。你不是说不怕？”

卷毛比尔看向她，眼神阴狠：“矿上女人少管——”

伊莱亚斯一把按住他的后颈，把他的脸压到桌面。卷毛比尔牙齿磕在木板上，血从嘴角挤出来。

“这里轮不到你挑人骂。”

屋里没人替卷毛比尔说话。

伊莱亚斯抓起卷毛比尔右手，把拇指按进印泥。卷毛比尔猛挣，椅脚刮出刺声。伊莱亚斯扣住腕骨往下一压，那点蛮劲被死死摁回桌上。

“说。”

卷毛比尔额头冒汗：“枪是我的。”

“昨夜谁让你去南街？”

“没人让。”

伊莱亚斯把缺口县印副本压到他眼前。纸边发冷，卷毛比尔的喉咙滚了一下。枪膛里那格空膛又轻轻响了一声，只有伊莱亚斯听见。

“谁让你去南街？”伊莱亚斯又问。

卷毛比尔咬牙：“后桌有人带话，说怀特会走南街。打碎徽章盒，街上就乱。工资箱夹层有人会取走。”

怀特猛地站起，肩口渗血：“谁带话？”

“贝汉手底下跑腿的。名字我不知道。”卷毛比尔不看他，“戴干净手套，付的是赌桌筹码。”

贝汉冷声道：“酒鬼供词不值钱。”

“所以枪值钱。”伊莱亚斯把卷毛比尔的左轮合上，放进旧证物箱，“枪、供词、锁片、回电和县印缺口都在这里。你要带走犯人，可以，先当着这条街写下你接走未备案文书、失枪、工资箱和谋杀证人的责任。”

贝汉嘴角绷住，手杖头在掌心转了半圈。

克拉拉把纸推过去：“我也可以替你记。字写得比你助手稳。”

贝汉看着那支笔，没接。

门外矿工开始说话，声音先低，后面越来越杂。卷毛比尔被骂得抬不起头。有人叫出工资箱，有人叫怀特别把枪交出去。艾达把工棚钥匙举起来，钥匙串撞出清响。

“矿工要看着这把枪留在墓碑镇。”她说，“谁拿走，谁先还我们的工资。”

怀特按着肩口走到门槛前。他看了伊莱亚斯一眼，又看门外的人。

“从今天白天起，临时副治安官伊莱亚斯·韦德暂管夜巡名单。”怀特声音有些哑，却压过门外杂声，“卷毛比尔的枪和供词，交由韦德保管，直到公开审查结束。”

伊莱亚斯取下墙上的夜巡名单，把自己的名字写在第一行。笔尖划过纸面，几顶脏帽子被人按到胸前，报童把号外重新展开。

怀特的老帮闲汤姆从墙边摸出一只旧钥匙圈，原本还想问要不要交给怀特，手伸到一半，又转到伊莱亚斯面前。

“夜巡灯柜的副钥匙。”汤姆清了清嗓子，“以前我喊你韦德先生，现在是不是得喊韦德副官？”

伊莱亚斯接过钥匙，铜环在掌心一沉。

“先喊能开门的。”

门外又有人笑。汤姆也笑，笑完马上把灯柜门推开，里面三盏夜巡灯全亮给街上看。灯一亮，卷毛比尔的脸更灰，贝汉助手的文书包也抱得更紧。

车轮声从街口压过来。

伊莱亚斯先把卷毛比尔的枪、供词和夜巡名单放回旧证物箱，箱盖扣住，手掌压在封条上。

贝汉的助手低声道：“先生，马车到了。”

两匹马停在治安官办公室外，车厢侧板挂着新的县权铜牌。车夫跳下，把一卷封蜡文书递给贝汉。

贝汉撕开封蜡，抬眼看向伊莱亚斯。

“图森刚补发的新任命。”他说，“这一次，有备案。”

马车门打开，里面露出一只空证物箱和一副新的监牢铁链。

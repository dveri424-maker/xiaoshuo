# 《恐怖片里的违规玩家》剧情大纲与写法定稿

本文档负责长篇结构、第一卷方向、章节节奏、正文风格和禁忌。世界观、系统、主角能力与长期因果见 `《恐怖片里的违规玩家》设定文档.md`；副本卡和章节卡见 `.claude/skills/meikong-horror-workflow/SKILL.md`。

---

## 1. 长篇结构总则

本书采用：

```text
可识别恐怖片现场
→ 原作/类型/系统误导
→ 原创隐藏规则
→ 死亡或污染代价
→ 林扬误判或救不全
→ 违规点动作化
→ 可见反制
→ 副本残响和长期后坐力
```

核心原则：

- 恐怖片现场优先，读者必须先看到鬼屋、录像、面具、娃娃、房契、病历、仪式、门牌、尸体、电话、地下室等可感物。
- 不写单纯电影打卡；原作或类型只是误导和期待来源，不是攻略。
- 不写普通玩家组织战争；长篇期待来自副本残响、债务、失败玩家痕迹和现实污染回流。
- 不把系统写成说明书；系统只能通过可见媒介影响副本。
- 林扬的反制必须落在物件、动作、站位、称呼、档案、照片、仪式或死亡触发条件上。

## 2. 副本最小结构

每个正式副本最终必须具备七件事，但章节不需要显性打卡全部内容。

### 硬必备

1. **强识别物**  
   让读者一眼进入类型期待，例如房契、旧照片、录像带、鬼娃、面具、病历、地下室、十字架、电话、校车、旅馆钥匙。

2. **误导**  
   可以是原作误导、类型误导、系统措辞误导或角色常识误导。

3. **原创隐藏规则**  
   真正杀人的不是表层怪物，而是模板之外的执行机制。

4. **代价验证**  
   死亡、不可逆污染、身份替换、记忆缺口、亲人误认、救人付债、身体异常等至少出现一种。

5. **可见反制**  
   林扬不能只解释规则，必须用物件、动作、站位、话术或系统措辞漏洞造成现场后果。

### 可延后但必须有

6. **林扬误判 / 救不全 / 欠债**  
   他可以赢，但不能全保；误判必须变成后续经验或债务。

7. **长期后坐力**  
   每个副本至少留下一个后续可回收物：错题本条目、被救者、怪物欠约、系统违规记录、奖励副作用、现实污染、失败玩家线索。

### 可选项

现实解释、队友配合、玩家痕迹、组织影子、原作角色深挖都是可选项，不要每次硬塞。

## 3. 第一卷方向

第一卷采用：

> 凶宅家庭诅咒 + 驱魔误导 + 伪造家庭档案 / 死亡债务规则。

### 商业入口

- 凶宅、驱魔、家庭诅咒是欧美恐怖最容易识别的入口。
- 老宅、房契、阁楼脚步、儿童涂鸦、家庭合照、地下室、神父、驱魔仪式能快速建立期待。
- 房契、照片、出生记录、驱魔记录天然适合承载“伪造身份关系”和“旧案债务转嫁”。

### 表层误导

- 房子闹鬼。
- 孩子看见不存在的人。
- 母亲隐瞒旧案。
- 神父或灵媒认为这是附身或恶灵占屋。
- 系统任务使用“保护”“驱逐”“存活”等原作式词汇诱导玩家。

### 原创隐藏规则

- 房子不是单纯杀人，而是在伪造家庭档案。
- 房契、旧照片、儿童画、出生记录、驱魔记录会把活人写入旧案债务。
- 驱魔仪式不是救人，而是可能给伪造关系盖章。
- 被写入档案的人可能短暂获得保护，也可能被迫承担前任死者、失踪者或施暴者留下的债务。
- 真正要解决的不是“驱鬼”，而是找出伪造签名、越权执行和死亡债务转嫁链。

### 开篇五章硬节奏

1. 第一章：出现强识别物或强类型场面，例如房契、儿童涂鸦、阁楼脚步、家庭合照错位、地下室门自己打开。
2. 第二章：出现规则死亡或不可逆污染，确认危险不是气氛渲染。
3. 第三章：林扬给出错误或半真判断，并完成第一次小型反制或救人。
4. 五章内：证明原作知识或类型常识不可靠，甚至可能害死人。
5. 第一卷中段：揭开房子正在伪造家庭档案，把活人写入旧案债务。

### 卷末成果

- 林扬查出档案签名漏洞或仪式盖章矛盾。
- 房子、恶灵或系统无法继续按原条款杀人。
- 林扬获得第一项核心长期资源，例如错题本正式成型、违规记录无法清除、被救者成为现实锚点。
- 留下债务或副作用，不能无损通关。

第一卷不能定义整本书。卷末应证明“恐怖片现场 + 原创隐藏规则 + 违规点动作化”成立，而不是把全书锁死在凶宅家庭主题里。

## 4. 分阶段扩圈路线

### 第一阶段：个人入场

林扬作为异常玩家进入恐怖片现场。重点是：

- 低恐惧反应；
- 现场证据；
- 原作误导；
- 第一次代价；
- 第一次可见反制。

### 第二阶段：残响累积

副本不再通关即归零。开始累积：

- 错题本条目；
- 被救者；
- 怪物欠约；
- 系统违规记录；
- 奖励副作用。

### 第三阶段：失败玩家痕迹

出现失败玩家遗物、错误笔记、残缺尸体、匿名留言。

作用：

- 证明系统筛过其他人。
- 给林扬提供错误案例。
- 制造“他不是第一个，但可能是最异常的一个”的长期期待。

### 第四阶段：现实回流

现实层开始出现副本污染：

- 现实档案变动；
- 手机时间异常；
- 医院记录缺页；
- 熟人记忆错位；
- 副本物件出现在现实。

### 第五阶段：高位规则痕迹

出现系统残响、结算异常源、副本校正者、规则维护痕迹、高位执行影子。

边界：

- 不写玩家公会争霸。
- 不写黑市交易升级抢主线。
- 不写官方部门权谋主线。
- 高位规则仍必须通过恐怖现场和可见物件落地。

## 5. 章节容量与追读节奏

章节是期待管理单位，不是信息容器。每章只围绕一个核心问题推进，接住上章钩子，解决一部分，再留下更具体的新问题。

章节功能类型：

- **钩子章 / 场景开启章**：强异常、强识别物、主角关键动作或下一场景入口。
- **展开章 / 探索章**：展开场景、人物阻力、第一批规则线索。
- **代价章 / 污染章**：用死亡、污染、替换、记忆缺口或付债验证规则重量。
- **反制章 / 关系章**：林扬基于现场信息完成小型反制、救人、压场或关系推进。
- **揭示章 / 转折章**：只揭示当前层答案，并抛出下一层更具体的问题。

执行规则：

- 单章最好只承接副本七件事中的一到两项。
- 系统提示、死亡代价、规则反转、奖励结算、新地图入口要分章接力。
- 如果一章无法用一句话概括，或读者需要同时记住三个以上新问题，应拆章或删减。
- 强钩子出现后要给读者发酵空间，不能立刻被更大的新钩子覆盖。

## 6. 正文风格

默认比例：

- 70% 严肃压迫。
- 20% 冷静拆解。
- 10% 黑色幽默。

### 恐怖写法

- 先写环境和日常物件的不对劲。
- 再写配角正常恐惧、误解或自我安慰。
- 让试探造成伤害、失踪、污染或身份异常。
- 林扬开口切开恐惧，但不把死亡写轻。
- 怪物、规则或系统回应他的挑衅。
- 林扬基于现场证据反制，不靠全知攻略。

### 对话写法

林扬可以荒诞，但每句怪话必须服务至少一项：

- 观察异常。
- 缩小规则范围。
- 标记违规点。
- 试探怪物或系统。
- 稳住配角。
- 压制恶人。
- 触发反制或规则反噬。

林扬的台词不能写成机械冷静。他可以在一句话里混用典故、粗口和贴吧抽象梗，但必须跟当前危险绑定。优先让典故自然融进比喻、反问、脏话和现场动作，而不是每次写成“原句 + 翻译一下”的固定公式。

- **引经据典**：用于压住荒诞感或给判断找落点，短句即可，不能变成知识科普。可以像“朝闻道，夕死可矣，老子早上打听到你家的路，晚上你就得死”这样粗暴改写，但不能反复出现“翻译一下”。
- **爆粗口**：用于惊吓、疼痛、濒危、鼓舞自己或打断恐惧读条，必须有压力来源。
- **贴吧抽象梗**：用于低声自我稳态、和周赫互相壮胆、或反制成功后的短暂泄压；不能只写“别送、人机、KDA、甲方”这种泛网感词，要把抽象落在照片、门牌、电话、录像、尸体站位等现场物件上。
- **黑色幽默**：要像人在极端压力下用嘴硬给自己撑住，而不是作者站出来抖包袱。

配角必须有回弹：质疑、反问、恐惧、隐瞒、专业判断、道德压力，不能只负责震惊。

### 黑色幽默边界

可以出现：

- 入场适应。
- 低危误会。
- 队伍磨合。
- 系统结算。
- 林扬压场后的节奏缓冲。
- 调查段的荒诞分类和反常比喻。

禁止出现：

- 死亡现场廉价玩梗。
- 恐怖未成立前先抖包袱。
- 儿童受害、身份替换、追杀、记忆缺口等关键代价段落插科打诨。
- 所有人都说同一种互联网腔。
- 与剧情物件无关的烂梗硬塞，尤其是只为“像网感”而出现的口头禅。

## 7. 禁止写法

- 电影打卡。
- 纯原创慢热开局。
- 第一卷看不出恐怖片类型。
- 原作角色只当彩蛋或工具人。
- 李昂低配。
- 游戏主播腔。
- 纯吐槽流。
- 主角只嘴炮不行动。
- “林扬说违规=问题解决”。
- 死亡现场玩梗。
- 规则说明书式正文。
- 无代价救全员。
- 系统万能黑箱。
- 玩家组织战抢主线。
- 把设定文档里的“副本结构模板”“商业爽点”“读者会觉得”等术语写进正文。

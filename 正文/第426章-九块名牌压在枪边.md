# 第426章 九块名牌压在枪边

九块名牌摆在市政厅长桌上，正好围住银柄左轮。

露丝、克拉拉、玛丽安、林月兰、娜奥米、莉迪亚、艾达、埃莉诺、塔莉娅。每个名字都刻得很细，仿佛刻牌的人在黑暗里贴着她们的呼吸数过。仿制钥匙、铅字、马哨、药瓶、学校牌、电报码、铁牌、金融印和盟约刀印分别放在牌旁，恶心得像一只偷来的手。

银枝纸条压在枪管下：请韦德先生选先救哪一位。

广场上的人不敢出声。敌人这一刀，比烧街、假票和棺材都阴毒。它不是让城乱，是让韦德离开桌子，让九条线同时觉得自己会被丢下。

露丝先伸手，把仿制钥匙拿起，看了一眼，轻轻放回去。

“齿口错三处。”她说，“偷听的人只听见响，没碰过门。”

克拉拉拿起仿制铅字，嗤笑：“这块字太新，没被我摔过。拿它冒充我的东西，连脾气都没有。”

玛丽安拈起马哨，吹都不吹，直接丢给身后的马队少年：“卡莱尔马哨没有这种便宜铜腥味。”

林月兰把假药瓶推远，声音冷淡：“瓶口没药渍。装样子的人从不救人。”

娜奥米指尖按在学校牌上：“字写反了。我的孩子不会犯这种错。”

莉迪亚看着那张电报码纸，眼底跳着火：“这是旧码。谁还用旧码威胁我，谁就等着被我绕死在线里。”

艾达把仿制铁牌捏得咔咔响：“轻得像饼干。砸人都嫌丢脸。”

埃莉诺拿起金融印仿件，白手套没有一丝抖：“印面没有压过账，东部穷得只剩样子了？”

塔莉娅最后看盟约刀印，刀尖一点，假印裂开：“没有旧誓味。只是树皮下的虫。”

一句句落下，广场的气重新回来。人们看见她们没乱，心就稳了一截。

伊莱亚斯把纸条收起，没有愤怒地拍桌，也没有冲向公馆。他站在九块名牌前，抬眼看向每个人。

“各守自己的门。”

露丝点头，转身把公馆钥匙串重新分组。第一门给女侍长，第二门给老马夫，第三门仍挂在自己腰侧。她当众把错齿假钥匙钉到登记册封皮上。

“今夜公馆只认人脸，不认钥匙声。”

克拉拉把假铅字塞进口袋：“报馆不停印。头版留一栏给银枝会，标题我已经想好：拿女人吓唬男人的人，连假票都印歪。”

几个报童立刻笑出声，跑向报馆。

玛丽安翻身上马，马哨真正响起，清亮得划破广场：“马队分三圈。内圈公馆，外圈车站，中圈医院学校。谁跑错路，我抽他。”

林月兰把假药瓶封进证物盒，提起药箱：“医院收伤，不收棺。灰粉中毒的人送侧廊，别让家属堵门。”

娜奥米把学校牌交给门队：“孩子今日读短课。所有窗户半开，退伍兵不上讲台，只守门。”

莉迪亚已经坐到临时电报桌前，手指敲得飞快：“三线同时报码。公馆安全，学校安全，医院安全。谁截我线，我让他收到一百遍自己的名字。”

艾达扛起真正的矿工铁牌，朝工人街矿工吼：“别堵市政厅看热闹！灰星仓水车归位，粮车归位，谁的老婆孩子在街里，谁自己站前面！”

埃莉诺打开账夹，把仿金融印压在假票证物旁：“基金柜继续兑付。小额真票先清，假票登记来源。东部若等我关柜，他们可以等到棺材烂掉。”

塔莉娅把裂开的假盟约印收进皮袋，望向城外灰星矿方向：“我去荒漠口。银枝会若借旧路进来，我割断它的影子。”

九个人各走一步，九条线像马缰一样绷直。不是谁替谁挡刀，而是每个人都把自己的门关紧，把自己的刀拿稳。

广场上的黑门头目看得发怔。一个老赌棍低声嘀咕：“这他娘的哪是软肋……”

艾达回头瞪他：“说完。”

老赌棍缩了缩脖子：“是根。是根，行了吧。”

伊莱亚斯把九块名牌收进一个黑铁盒，盒盖没有盖。他让所有人看着那些名字，又把银柄左轮放在盒旁。

“名字不躲。”他说，“谁敢碰，谁就留下手。”

消息沿九条线同时回到广场。公馆那边，露丝换过钥匙后，两个假送货人被老马夫挡在门外，包裹里藏着细锉和蜡模。报馆那边，克拉拉的新号外刚上街，就有人撕报，被报童和女工围在墙角，逼着当众念完整篇辨票图。马队在北街截住一辆空车，车底藏着公馆侧门草图。医院侧廊收了三名灰粉呛伤的人，林月兰让他们先漱口再供话，供出了棺材仓库的旧路。学校地下教室里，娜奥米让孩子们照常读课，一声哭闹都没放出门外。

莉迪亚的电报码一条条落在桌上，艾达每听一条就冲矿工们吼一声，让工人街别乱跑。埃莉诺坐在基金柜前，照样兑票，故意把窗口开得更亮。塔莉娅从荒漠口传回短讯：银枝会探路骑手退了，留下一截带血的树枝。

九块名牌还压在枪边，可九个人没有一个被牌子拖住。广场上的人看着电报纸越堆越高，终于明白敌人挑错了东西。名字不是绳子，名字是门牌。门牌在自己人手里，就能守门。

乔纳带着骑警从车站赶回，身上还带烟味：“公馆外暂时没动静。可主章柜那边少了一个守柜账房，门锁没撬，像是拿真钥匙进去的。”

埃莉诺脸色一冷：“基金主章。”

露丝立刻翻登记册：“主章柜钥匙只有三把。我的，韦德先生的，还有签署桌备用封钥，备用封钥昨夜封在市政厅证物柜。”

莉迪亚抬头：“证物柜外线刚断了一息。”

伊莱亚斯眼神沉下去。

敌人把九块名牌送到桌上，不是为了让他选择女人，而是为了让所有眼睛都盯着公馆。真正的手，伸向基金主章。

他转身走向市政厅侧室。露丝、埃莉诺和乔纳跟上。证物柜门还关着，封蜡看似完整。伊莱亚斯蹲下，看见封蜡边缘有一圈极细的热痕，像用银针烫开又压回去。

露丝吸了口气：“内行。”

伊莱亚斯拔出备用封钥。钥匙还在，但齿口沾着一点黑粉。侧室窗台下，有一枚木屑，木屑上刻着半截银枝。

远处车站方向忽然响起调度钟，三短一长。

莉迪亚冲进来：“有刺客从主章柜跑了，没偷走整枚章，只刮走章面半寸！乔纳的人追到货台！”

埃莉诺声音第一次失了平稳：“半寸也够伪盖一份转让书。”

伊莱亚斯把九块名牌盒盖扣上，交给露丝。

“锁进公馆，不藏，挂在议事厅门内。”

露丝接过盒子，钥匙串响得像短促的誓言。

他还没催马，露丝忽然叫住他。她没有说小心，只把主章柜被烫开的封蜡边角递给他。那点红蜡薄得像血皮，贴在一张白纸上。

“带着。”露丝说，“回来对痕。”

埃莉诺也把一小枚空白封签塞进乔纳手里：“章屑若追回，先封，不许任何银行手套碰。”

莉迪亚从电报桌旁抬头，指尖还压着键：“我给车站锁三短一长，听到第二遍就是东线车要动。你别让他们把车开出去。”

玛丽安在马上笑了一声：“他若让车跑了，我替你追。”

“你守你的圈。”伊莱亚斯说。

玛丽安扬眉，却没有顶嘴。她知道现在每个人都不能离自己的门。九块名牌的铁盒被露丝抱回公馆议事厅，盒盖锁响传到广场，像给他送行。

台阶下，一个刚才还发抖的商会学徒忽然把帽子摘下，冲公馆方向鞠了一躬。他不是向哪一位女士献殷勤，而是看见九条线都没断，终于明白自己昨夜能活着守住铺子，不是靠东部银行那张纸。旁边的矿工拍了拍他肩膀，两人没说话，却同时转身去搬水桶。

伊莱亚斯这才催动礼拜日。

伊莱亚斯翻身上马，银柄左轮在晨光里一闪。

“我去拿回那半寸。”

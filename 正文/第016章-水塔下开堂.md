# 第016章 水塔下开堂

中午，长桌摆到水塔脚下。

桌子从酒馆借来，露丝让伙计擦了两遍，刀痕还在，威士忌渍也没全擦掉。霍布搬来两只木箱，箱盖一翻就是证物台。乔纳把东西摆上去，没摆得太齐，旧支洞钥匙压着出入簿誊本，红盐包挨着工程绳碎段，车厢木刺、棕色袖套、缺齿齿轮铜扣各占一角。

林月兰没来。药铺伙计送来一张验伤条，放下就要走，又被克拉拉叫住按了手印。纸边是细瘦签名：勒死在前，悬挂在后，死后肋侧刮伤，木漆残留。

伙计压低声音：“林姑娘说，字别沾水，证物别沾铁路。”

露丝在酒馆檐下听见，放下咖啡杯。“这医嘱不错。”

克拉拉站在桌子右侧，笔记本摊开，油墨味被晒热的木头味压了一半。她不坐，笔尖一直搁在纸面上，谁抢话，她就先记名字。

镇民围了一圈，不整齐。矿工挤在前头，洗衣妇抱着木盆站在阴影里，帽店姑娘从裁缝铺门边探头，两个孩子被母亲拽着衣领，还想看桌上的红盐。几个卡莱尔牧场骑手站在外圈，玛丽安没进来，只骑马停在街对面，手套扣着缰绳，栗马尾巴一下下扫尘。

范恩最后到。

他带着基恩、书记员和六个铁路私兵。基恩换了干净外套，袖口空着，手套扣得更紧。私兵一到就往水塔左侧挤，把一个卖苹果的女人挤得篮子歪了。苹果滚出两只，孩子想捡，被母亲一把按住。

乔纳没骂人，只往证物桌前横了半步。内特和山姆也挪过来，把红盐木箱护在身后。

一个私兵伸手去碰车厢拓印。

伊莱亚斯用一枚银元压住纸角。

“看可以，手留在自己袖子里。”

私兵抬眼。伊莱亚斯靠着桌边，黑色大衣被日头晒出一点尘灰，银柄枪安静贴在腰侧。银元在纸上转了半圈，停住。那私兵把手收回去，改去扶皮带，嘴里低声骂了半句。

克拉拉在页边写：证物未离桌。

露丝让莉莎端来一壶水，放在长桌另一头。谁想说话，先看那壶水，再看头顶的水塔。卖苹果的女人把篮子放到孩子脚边，没有再退。

范恩扫过长桌，笑得体面。

“韦德先生，你在扮法官？”

伊莱亚斯抬了抬下巴。“我坐着了吗？”

书记员忍不住道：“法庭不是杂货摊，东西摆齐就能卖。”

露丝慢慢接了一句：“杂货摊至少标价清楚。”

几个矿工笑了一下，笑声很短，看见灰蓝制服后又闭了嘴。

范恩没看她。他盯着克拉拉的笔，又看乔纳身前的证物。“这是铁路的人，铁路的车，铁路的工地。要查，也该回铁路营地查。你当街点名，是想把别人的罪名扣到公司头上。”

伊莱亚斯拿起工程绳碎段，绳头还有油腻味。“霍兰脖子上的绳结，不是牧场活扣。乔纳认过，霍布也闻过。煤灰和机油从绳芯里渗出来，死者袖口也有。”

霍布摸着胡子。“牧场绳子没这个味，除非牛开始吃煤。”

孩子扑哧一声笑，被母亲拍了后脑勺。

伊莱亚斯放下绳子，拆开红盐包。细红灰落在白纸上，被风吹动一点，又被银元压住。“死者靴底的红盐，和旧支洞门槛上刮下来的这一撮对得上。钥匙是范恩先生交的，内栅门却有铁路私兵拦着。”

范恩道：“阻拦是出于安全。”

“死人不会动。”伊莱亚斯看着他，“你们怕的是桌上的东西。”

水塔横梁在上方轻轻响了一下。有人抬头，有人没抬，老矿工咬着烟斗骂了句脏话。

伊莱亚斯把车厢拓印摊开。纸上两道刮痕很短，旁边压着林月兰的验伤条。“夜车乙二地板上的刮痕，对得上死者肋侧死后伤。车厢内壁有我昨夜留下的墨记，今早铁路想调走，没调成。木箱角黑漆和旧支洞门槛木刺同一桶漆。”

霍布用小刀从两片黑漆木屑上各刮一点粉末，撒到白纸上。两撮漆粉都黑里泛灰，煤油味被太阳一烤，更冲。

“同漆。”霍布说，“两边都沾煤油味。”

书记员急了：“你承认夜闯车站？非法所得证据不能——”

“黑河镇没有法院。”伊莱亚斯打断他，“但有眼睛。巡回法官来了，自会看哪辆车、哪道门、哪只袖扣。”

克拉拉的笔尖刮过纸面，没有抬头。

伊莱亚斯把齿轮铜扣放到桌面中央。铜扣缺一齿，旁边是夜车厢里取出的铜屑。霍布伸手按住，用钳尖轻轻一推。

咔。

铜屑贴回缺口。声音不大，桌边几个人都听见了。

“棕色袖套，齿轮铜扣，缺齿铜屑来自夜车乙二。洗衣房有人给基恩工程师补过袖套。证人姓名不公开，证物公开。”

基恩开口：“很多工程师有这种扣。”

伊莱亚斯看向他。“把你的拿出来。”

基恩手指一下攥紧。

范恩插话：“任何雇员没有义务在街头接受羞辱式搜查。”

“所以我没搜。”伊莱亚斯点头，“我请基恩先生解释，合同副页上的E.K.、出入簿涂改下的K、工程处封存令，还有他袖套线头为什么全指向他手里那枚印章。”

风从水塔架下穿过，吹得验伤条纸边发响。基恩嘴唇动了动，没出声。

范恩上前半步，手杖尖点进尘土。“够了。韦德先生，你没任命，没徽章，连镇民会议那张破纸都没有。你拿几件脏东西逼我的人开口，还想让整座镇子跟着你起哄？”

伊莱亚斯低头看那根手杖。

“说得对。”

克拉拉的笔停了一下。人群里也有人愣住。

伊莱亚斯抬眼，帽檐下的灰眼亮了一下。“我没有徽章。所以我不判他。我只把东西摆在黑河镇喝水的塔底下，让大家看清楚，谁把死人挂上去，谁想拖走尸体，谁在旧支洞门口拦人。”

他指向长桌。

“你想拿文件否认这些证据，可以。先把红盐塞回靴底，把袖扣塞回车厢，把出入簿那道黑墨抹干净，再把林姑娘的验伤条改成活人自己上去挂绳。”

这话落下，没人急着接。连书记员都去看范恩的手杖尖。

霍布往桌边站近一步。卖苹果的女人把篮子放稳，手还护着孩子，却没退。格蕾丝站在裁缝铺门口，报纸折在手里，眼睛从范恩脸上移到伊莱亚斯胸前的枪套，又抬到他帽檐下那张俊美冷硬的脸。她很快低头，把报纸边捏皱。

乔纳把旧支洞钥匙放到桌上，压在出入簿誊本旁。

钥匙落下，基恩眼皮跳了一下。

伊莱亚斯把钥匙推向他。

“工程师先生，既然都是巧合，就当着大家的面，用这把钥匙开旧支洞内门。让我们进去看一眼。什么都没有，我请范恩先生喝最贵的威士忌。”

露丝在檐下补了一句：“先付账。”

这次有人笑出了声，没立刻闭嘴。

基恩盯着钥匙，手套指节一下一下收紧。

范恩压低声音道：“基恩先生不需要回答荒唐要求。”

伊莱亚斯把钥匙收回掌心。

“他已经回答了。”

克拉拉写下这句。写完抬头，正看见基恩额角的汗滑进干净领口。

水塔下没有槌声，只有水壶里的水轻轻晃。

范恩转向人群，语气重新平稳。“诸位，别让怕意替你们认案子。巡回法官来以前，东西交铁路封着，镇民会议派人看着。今天散了。”

人群后，面粉铺老板费奇擦了擦汗。他是镇民会议委员。听见“镇民会议”，脚尖往后挪了半寸。

伊莱亚斯把霍兰的裂表放到长桌最前面。表盖打开，指针停在凌晨三点十七分。

“费奇先生。”

费奇一抖。“我？”

“你家的面粉靠铁路运。”

费奇脸白了。“这当然——”

“你家的水也从水塔打。”

费奇闭嘴。

伊莱亚斯把红盐包、铜扣、袖套、验伤条、出入簿誊本和车厢拓印推到桌沿，最后按下合同副页。

克拉拉第一次看见副页全貌，笔尖差点戳破纸。

旧支洞仓。夜车乙二。样本转运。封存费。E.K.

老矿工挤到前面，盯着“样本”两个字。

“什么样本？”

基恩的汗落得更快。

伊莱亚斯看着他。

“下午还长。”他说，“慢慢问。”

水塔横梁又响了一声。

这一次，没人散。

# 第三章 玩家

林扬看着住户名册上那张没有眼睛的证件照，第一反应不是跑。他先看照片边缘。照片是贴上去的，不是印上去的，胶水还没干，左下角微微翘起，露出一层灰白色纸毛，像旧报纸泡水后晒干的质感。照片下方有三行字。

`姓名：林扬。`

`照片缺失。`

`本人应声：未。`

“应声？应你奶奶个腿”。林扬在心里很轻地骂了一句。

这叫把卖身契贴到你脸上，还问你为什么不按手印。门内那个女人仍然站着，手里捧着住户名册，笑容温和，耐心也温和。她没有扑出来，也没有伸手抢人，只是等着，等林扬说话。

林扬退了半步，鞋跟仍然留在门槛外。他没有回那句“回来了”，也没有说“我是林扬”。门牌下方的打印纸被夜风吹得微微发抖，上面“名单无此人”四个字正压着铜牌新浮出的“临时住户：林扬”。墨迹没有消失，但边缘正在变浅，像被门内的潮气一点点舔掉。

女人低头看了名册一眼。册页上“本人应声：未”后面的空白处，缓慢渗出一小截黑线。那黑线没有写成字，只像一根探出来的针，试图找到能落笔的地方。林扬把透明文件袋往门牌下又压紧了一点。

“原名单无此人。”他说。

这句话不像咒语，也不像狠话，更像一个大学生在旧楼门口和一堆纸较劲。可效果很直接。名册上那截黑线停住，慢慢缩回纸里。女人脸上的笑容终于淡了一点。

“进来拍照。”她说。

“不拍。”林扬说。

“住户都要照片。”

“我不住这儿。”

女人抬起头。她的眼睛里没有瞳孔，只有两张很小的黑白合影。合影里的人站在三号楼门口，所有脸都朝外看。

“你到楼下了。”她说。

“外卖也到楼下。”林扬说，“你把外卖写进住户名单吗？照你这算法，全沪城骑手都是你家亲戚。”

女人安静了。二楼灯光开始闪烁。

门厅还是白天那个门厅，水泥地，老式信报箱，墙上贴着褪色的消防示意图。赵明宇的尸体已经被运走，拖痕却还在，灰白色，直直通向楼梯口。楼梯上方挂着几张旧照片，照片里都是三号楼住户合影。

每一张照片角落里，都有赵明宇。有的站在楼门右侧，有的坐在楼梯口，有的趴在窗户后面。姿势越来越低，脸越来越糊。最后一张里，他只剩一只手，从门缝里伸出来，指甲缝全是灰。

林扬终于明白了一点。赵明宇不是被人摔死的。他大概也见过自己的照片，也被叫回了这栋楼。他想逃，逃到门厅，离门口只差几步，最后还是没出来。至于为什么没出来，林扬还不知道。他只知道这女人手里的册子很要命，墙上的照片也不正常，而自己现在最好不要承认“我是林扬”，更不能顺着那句“回来了”往下接。

女人又翻了一页。林扬那张无眼证件照开始变清晰。原本模糊的脸颊、头发、衣领，一点点贴合他现在的样子。只有眼睛的位置仍然是空的，像两个等着被填上的洞。

林扬不是不急。他掌心已经出汗，手腕也在发僵。眼前这东西明显不是能讲道理的类型，但它偏偏又没有直接扑出来。它在等字，等照片，等他说话，等他走进门。林扬不知道这算什么，只能按普通人的笨办法来：它要什么，就先别给什么；它靠什么，就先弄坏什么。

“稳住。”林扬低声说。这句话是说给自己听的。手抖一下，照片墙上就得多挂一个傻逼。他摸到口袋里的旧“作废”章，木柄边缘磨得发亮，底部橡胶裂了一点。正常情况下，它只能给纸盖个红字，不能驱鬼，也不能打架。但眼前这东西把人名写在纸上，把照片贴在纸上，还拿着一本册子站在门口装宿管。既然它这么喜欢纸，那就拿纸上的办法试试。林扬把印章藏在掌心，另一只手捏住打印纸，向前半步。女人的笑容重新出现。

“林扬。”她轻声说，“回来。”

“你这照片不合格。”林扬说。

女人翻册页的手停住。林扬举起打印纸：“一九七九年原名单里没有我，门牌上也没有我。你现在临时往上贴照片，连眼睛都没贴出来，属于盗版证件照。名不正则言不顺，你这破楼连我名字都没写明白，就想把老子塞进相框里，做梦。”

女人歪了歪头。她的脖子发出纸页折叠的声音。

“本人在场。”她说。

“本人没同意。”

“已到楼下。”

“楼下不算进屋。”林扬说，“不然全沪城外卖员都是业主。”

门内那些旧照片一张接一张转向门口。照片里模糊的人脸贴近相纸边缘，像在围观一次很久没有发生过的离谱争执。林扬继续说：“你要拍照，可以。把册子递出来，我看看拍得像不像人。”

女人笑了一下。

“进来看。”

“你拿出来。”

“住户进楼。”

“管理员下楼。”

女人的笑容消失了。门内的温度突然降下去。女人手里的名册哗啦啦翻页，翻动速度快得像有风从书脊里钻出来。册页上密密麻麻全是名字，有些名字旁边贴着照片，有些名字被黑线划掉，有些照片上的脸已经被灰色墨水涂满。赵明宇在其中一页，林扬也在下一页。女人抬起手，门内所有照片同时脱离墙面，像一群黑白色的蛾子，朝门口飞来。

林扬把打印纸往门牌上一拍，用胶带把它贴死在铜门牌下方。打印纸上的“名单无此人”正对着门牌新浮出的“临时住户：林扬”，两行字像互相咬住的牙。照片飞到门口时停了一下。

停顿很短，但够了。

林扬侧身跨到门槛边缘，没有完全进楼，只把右手探进去，一把按住名册翘起的那页。

女人的手冰冷得不像手，更像泡过水的纸。她猛地抓住林扬手腕，力气大得惊人。林扬半个身体被往里拖，鞋底擦过门槛，眼前一阵发黑。他左手死死扣住门框，指节磕在粗糙墙皮上，疼得他低声骂了出来。

“本帅哥不喜欢大妈”，林扬咬牙，“操，劲还挺大。”

女人低声说：“回答我。”

林扬手腕被抓得发疼。皮肤下面像有细小纸片往里钻，试图找到血管和名字。名册上那张无眼证件照的边缘开始贴平，胶水要干了。

疼痛让他的视野边缘开始发白。林扬咬住牙，低声骂道：“别补了，真别补了。身体发肤受之父母，父母之命媒妁之言，真喜欢我，给我下聘礼，我没说不能入赘。”他说完，抬起藏着印章的手。女人似乎察觉到什么，猛地合上名册。林扬等的就是这一瞬间。

名册合拢时，夹住了他的打印纸边角。打印纸上有原始名单，有他亲手写的“名单无此人”，也有校史馆打印出来的时间戳。

林扬一把按下“作废”章。不是盖在自己名字上。他盖在“本人应声：未”那一行上。红色印泥在册页上炸开。

`作废`

两个字歪歪扭扭，却红得像刚流出来的血。

女人发出一声尖叫。那声音不像人，更像很多张照片同时被撕开。门厅里的灯管一根接一根爆裂，墙上的合影疯狂抖动。赵明宇的照片从相纸里凸出来，像有东西在里面挣扎着往外爬。林扬没有停，他用尽全力把名册往门外一拽。女人半个身体被拖到门槛前。她脚下没有影子，裙摆下面也不是腿，而是一叠叠发霉的纸页。纸页上写满住户姓名，有些名字已经糊成黑块。

“别侮辱打印机。”林扬额头冒汗，像是在反驳自己脑子里某个荒唐比喻，“打印机至少卡纸会提示。你这属于不告而取，不告而取谓之贼。偷脸的贼，欠盖。”

女人的脸迅速变化。一会儿是普通宿管，一会儿是赵明宇，一会儿是陌生老人，一会儿又变成林扬那张无眼证件照。她试图把自己的脸稳定在林扬脸上，但“作废”两个字像一块烧红的铁，压在那行“本人应声”上，让她始终差一口气。林扬抬脚，踩住门槛外的铜门牌边缘。

“本人不应。”他说，“照片不采用。”

女人尖声道：“林扬！”

“叫魂也要讲基本礼貌。”林扬喘着气，“别搁这儿一口一个本名。君子不立危墙之下，你这破楼这么老，还想招待我住下”

林扬把第二个“作废”章盖在自己的无眼证件照上。

这一下，门厅里所有照片同时白了一瞬。赵明宇的身影从合影角落里被硬生生弹出来，摔在门厅地面，变成一团灰。那团灰里露出一张被啃掉照片的学生证，又迅速碎开。女人抓着林扬的手松了。她想退回楼内，但名册被林扬拖在门外，门牌下方的打印纸死死贴着，像一张迟来的拒收单。她的身体开始从门槛处分层，一层是纸，一层是照片，一层是发霉的布料，一层是黑色的手印。

林扬看见她胸口有一张旧工作证。

`北区三号楼管理员：陈桂兰`

照片上是一个中年女人，眼睛已经被灰色墨水涂掉。她不是这栋楼本身，她只是被这栋楼留下来的那个人。

林扬伸手抓住那张工作证。女人猛地抬头，终于不笑了。

“住户必须回楼。”她说。

“你也该下班了。”林扬说，“这么喜欢上班，你这个工贼”

林扬把工作证从她胸口扯下来，反手盖上第三个“作废”章。

红字落下。三号楼里传来一声沉闷的响动，像整栋楼的柜子同时倒塌。女人的身体骤然散开，变成无数发黄照片。照片在门口盘旋了一圈，又被看不见的风吸回楼内，贴回墙上。这一次，墙上合影里没有赵明宇，也没有林扬。林扬坐在台阶上，喘了很久，才低头看那本被拖出来的名册。

名册还在，但封皮已经裂开，里面大部分页面空白。只有第一页留下两行字：

`林扬：未入住。`

`陈桂兰：作废。`

林扬手腕上有一圈青黑色指印。指印下面，皮肤里有细小的纸纹，像被某种东西短暂按住过。他皱了皱眉，把名册合上。

下一秒，周围的声音突然低了下去。

雨后的虫鸣、远处宿舍楼的吵闹、保卫处车辆里的广播声，全像被一只手按进水里。林扬眼前浮现出一片冷白色文字。

【检测到候选者在无提示状态下解除初级异常事件：北区三号楼】

【异常执行人：陈桂兰，已清除】

【受害者：赵明宇，已释放】

【候选者：林扬】

【玩家资格生成中……】

【资格生成完成】

【基础特性：低恐惧反应】

【临时标记：违规玩家】

【新手任务将在二十四小时内开启】

【提示：请勿拆取副本道具】

林扬看完，沉默了两秒。

“我去，这是啥？入职通知？”他低声说，“看起来不像正经单位，而且不让带走办公用品。”

他把那枚沾着红印泥的作废章收进口袋，又捡起地上裂开的名册。

白字忽然闪了一下。

【警告：副本道具不可携带】

名册封皮上的霉斑正在慢慢消失，像要从现实里退回去。

林扬想了想，撕下第一页。

【警告：副本道具不可携带】

第一页也开始变淡。

林扬又撕下一角，塞进自己手机壳背面。

【警告：副本道具不可携带】

那一角纸停住了。

白字卡了一下。

【异常携带判定中……】

【判定失败】

【记录：候选玩家存在违规倾向】

林扬笑了一下：“小气。”

三号楼门内，已经熄灭的二楼灯忽然又亮了一盏。

不是刚才那种阴冷的灯光。这一次，灯下站着赵明宇。他的脸依旧模糊，身体也淡得快要看不见。他低头看了看自己空荡荡的手，又抬头看向林扬，嘴唇动了动，像是想说谢谢，也像是想提醒什么。最终，他只指了指林扬的手机。手机屏幕自己亮起。

一卡通小程序里，那条门禁记录变了。

【北区三号家属楼门禁记录：林扬，23:13，未入住。】

下面多了一行。

【下一次地点：待定。】

林扬看着那行字，慢慢把手机收起来。铜门牌上的字迹恢复成了原样，只是最下方多了一道红印，歪歪扭扭的“作废”二字嵌在铜锈里，像一道不肯愈合的伤口。

他拎起放在台阶边、已经彻底冷掉的鸡排饭，往宿舍方向走。人在入职前，至少应该把晚饭吃完。

身后，三号家属楼的门无声关上。门缝里飘出一小片发黄的照片碎屑，落在雨后的台阶上，碎屑背面有一行极淡的字。

`玩家已确认。`

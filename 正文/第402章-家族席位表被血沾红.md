# 第402章 家族席位表被血沾红

半张家族席位表摊在公馆长桌上，血从纸角渗开，像红墨写错了名字。

露丝拿来银星封袋，指尖还缠着布。她没有碰血，只把断开的钥匙环放在纸边。钥匙环上少了一枚内门副钥匙，断口新亮，像刚被细钳咬过。

克拉拉站在桌侧，铅笔抵着纸，没写。她难得没抢第一句话。前厅里还有女侍在清碎玻璃，门外马蹄、枪托、低声咒骂混在一起，公馆像一只刚被刀划醒的兽。

“别先问谁丢了钥匙。”伊莱亚斯说。

几个女侍僵在门边，眼睛红着。

露丝抬眼，看了他一瞬，手里的登记笔才重新落稳。

“先问血往哪儿走。”他把银柄左轮放在席位表上方，“谁急着让公馆互咬，谁就在外头等我们乱。”

埃莉诺披着深色外衣坐下，白手套只戴了一只。她把金融印章盒推到桌边，印盒铜扣上有一道被刀尖挑过的痕。

“这张表不是摆设。”她声音很轻，“东部银行认表上的席位，认受益顺序，认签字和钥匙附录。缺半页，他们就能说韦德家族自己都没写明白。”

艾达在门口骂了一句：“狗屁。人还没死呢，他们先抢椅子？”

“他们抢的不是椅子。”露丝把断钥匙环压进证物盘，“是门。”

林月兰提灯靠近。她蹲下查看纸角血迹，又看了看长桌下那条细黑血线。红线药瓶贴在她胸前，瓶内药液轻轻晃。她用银针挑起一点血，放在白瓷片上。

“不是一个人的血。”她说，“纸角的血新，窗边那滴旧。旧血里有灰星冷粉，伤口会发麻，能让人跑得久一点。”

“所以中枪的矮子还跑得动。”乔纳冷着脸，“我带人追到花园墙，他翻过去了。墙外有人接，马蹄包了布。”

玛丽安把马哨摔在桌上，眼神锋利：“马蹄包布也会压草。后巷外有两匹轻马，往学校街绕了。不是牧场马，是戏班子拉车那种短腿马。”

克拉拉终于落笔：“刺客骑戏班马逃向学校街。”

娜奥米脸色一白，却没有乱。她把学校门队册抱紧，指节发白：“学校今晚留了退伍兵。孩子们不在训练场，只在宿舍。”

伊莱亚斯看她：“你带路。别一个人跑。”

娜奥米点头：“我的门队听我的口令。你到校门前，也要卸枪。”

屋里静了一下。

艾达瞪眼：“这时候还卸枪？”

娜奥米没有退，眼睛却看着伊莱亚斯：“孩子听得见枪套声。规矩一破，明天他们就怕自己的门。”

伊莱亚斯解下备用短枪，放到长桌上，只留银柄左轮在腰侧。

“到门前卸。”他说，“出门后再戴。”

娜奥米肩膀松了半寸。

莉迪亚抱着电报码本从窗边转过来。她的袖子上沾着灰，脸颊有一道擦伤。“我截到一组旧码。不是刚发的，是三年前电报局弃用的练习码。有人故意把它夹在钟声里，让我以为是孩子门队求救。”

“谁知道旧码？”伊莱亚斯问。

莉迪亚咬了下嘴唇：“老电报员、东部铁路局、还有我以前教过的一批学员。”

她说完，自己先抬头：“我查。我不躲。”

伊莱亚斯看了她一眼，没有安慰。

“你查线。”他说，“线断之前，别让任何人替你按键。”

莉迪亚把电报码本合上，眼睛亮得厉害。

塔莉娅站在花园侧门，盟约刀副印贴着掌心。她没有进屋，像不愿让鞋底的泥碰到席位表。过了片刻，她低声说：“血味里有荒漠废站的纸浆味。不是旧帮派的手。”

“银枝会。”埃莉诺说。

伊莱亚斯伸手，把半张席位表拿起。纸面在灯下泛冷，血迹边缘有一小块银灰色粉末。墙外夜风吹进来，烛火忽然压低，桌下像有一枚看不见的马刺轻轻碰地。

门边那个被活捉的刺客猛地哆嗦，牙关咯咯响。

“我只是拿纸。”他崩着嗓子，“我没杀人！”

伊莱亚斯转头。

刺客被乔纳按跪在地，左肩中枪，脸上全是冷汗。刚才怎么逼都不说话，现在却像被死人贴着耳朵吹气。

“谁要纸？”伊莱亚斯问。

刺客喉咙发紧：“树……树徽的人。银枝会。他们说韦德家里的女人各拿一条命根子，只要撕掉名字，就能让你们自己抢。真的，我没碰孩子，我只开门。”

艾达一脚踹过去，被乔纳拦住。

伊莱亚斯走到刺客面前，蹲下，把复制钥匙放在他眼前。

“齿口从哪儿来？”

刺客看见钥匙，眼睛缩成一点：“黑市匠人只磨外齿，内齿是……是钟房给的谱。钥匙声谱。我们照声音磨。”

露丝脸色冷下来。

“第三道门的谱呢？”伊莱亚斯问。

“缺一齿。缺的那一齿在学校训练场。”

娜奥米猛地攥紧门队册。

刺客像怕自己说慢了，急声补：“他们把第二页埋在那里，说炸开以后，就说韦德公馆的女人争席位，把学校也拖下水。火药不是我埋的！我他妈只是拿钱开锁！”

艾达冷笑：“听见孩子两个字才知道怕，晚了。”

伊莱亚斯起身，枪口往下一压。

刺客不敢再喊。

“乔纳，活口锁进马房，嘴里塞布，不许死。”

“是。”

“露丝，公馆内门换铁闩。少的那枚副钥匙，挂空位，别补。”

露丝抬笔：“让他们看见空位？”

“让他们以为还能用。”

露丝唇角轻轻一动：“明白。”

克拉拉撕下一张空白纸，递给伊莱亚斯：“我写什么？”

“写公馆遇袭，刺客被捕，学校无伤。”

“学校还没查。”

伊莱亚斯把席位表残页压进封袋，抬眼看她。

克拉拉骂了一声，把铅笔削尖：“行，先把黑锅踢回去。”

埃莉诺把印盒收紧：“席位表金融副本在我账夹里，缺的是签署原页。东部律师明天会来咬。”

“让他来。”伊莱亚斯说，“今晚先拆炸药。”

他把复制钥匙、断钥匙环和席位表残页分成三只封袋。第一只交给露丝，挂在公馆钥匙板下方，不许进铁柜；第二只交给克拉拉，让她只写复制钥匙，不写内门顺序；第三只由乔纳带在身上，银星骑警押送到学校街。

埃莉诺抬手拦住一个想替她披外衣的女侍，自己扣好账夹：“东部明早会问我，席位表缺页后金融副本还算不算数。”

伊莱亚斯看着她：“你怎么答？”

埃莉诺把第九钥匙轻轻一拨，钥匙撞在账夹铜角上：“我会让他们先承认，今晚有人为一张不算数的纸流了这么多血。”

克拉拉笑了一声：“这句话别浪费，留给我在报纸上杀人。”

娜奥米已经走到门口，听见“学校街”三个字，脸上的血色又退了些。伊莱亚斯把备用短枪推给乔纳，自己只拍了拍腰侧银柄左轮。

他们离开公馆时，天边没有亮色。门廊下那名女侍还举着断钥匙环，手臂发抖也不敢放下。几个黑门旧人被艾达赶到街对面，嘴里骂骂咧咧，却没一个敢越过公馆灯线。露丝站在门内，把新铁闩一枚枚扣上，每扣一下，屋里的人心就稳一寸。

娜奥米走在前面，手里攥着学校门队口令牌。伊莱亚斯跟在她半步之后，腰侧银柄左轮在黑暗里沉默。玛丽安马队散开封街，艾达带着两个矿工扛来铁钩，林月兰提药箱，莉迪亚抱电报箱边走边听线，塔莉娅在最后，指尖抹过墙角血痕。

公馆门廊上，露丝把断钥匙环挂到灯下。

少掉的那一环空着，随风轻轻撞门。

学校街尽头，旧训练场的靶墙后，忽然亮起一线不该有的蓝火。娜奥米停住脚步。

伊莱亚斯越过她，重新系上枪带。

“门前我卸枪。”他说，“门外不用。”

娜奥米没有笑，只把口令牌举高，让跟来的退伍兵都看清。伊莱亚斯走在她身侧，没越过她半步。公馆到学校这段街，第一次由学校门队在前面开路，马队和骑警反倒压在后方。街边几个被惊醒的家长探出头，看见娜奥米手里的牌，才敢把门开宽一点。

# 第284章 马车房里不跳舞

马车房里没有乐声。

外面水晶灯下的小提琴还在拉，隔着墙板传来，薄得像一层假笑。马车房里只有马喷鼻、车轴轻晃、油灯火苗被风压低的声音。

红灯下第三辆马车半开着门，车轴上的白漆被泥蹭掉一截。埃莉诺站在车旁，背抵着车门，礼裙裙摆被木屑划出一道暗痕。她两只手仍戴着白手套，左手按着一只棕色账夹，右手被一个铁路枪手扣住腕口。

账房躲在马车后，脸上全是汗。

“夫人，”他尖声道，“范恩先生只要账夹。您把东西交出来，没人会为难您。”

埃莉诺抬眼，冷得像冰面：“你们把门闩落下、油灯移到干草旁，再说不为难？”

另一个枪手把火柴盒在掌心掂了掂：“火是意外。舞会夜，马车房油灯翻倒，多常见。”

伊莱亚斯进门时，正好听见这句。

他没有拔枪。

门口的枪手转身最快，手刚摸到枪柄，伊莱亚斯已经跨过门槛，左手扣住他的手腕，右手按住他肘弯。骨节错开的声音被马蹄一跺盖住。枪手整个人翻过半圈，脸朝下砸进草料堆，枪还在套里。

第二个枪手抬枪。

伊莱亚斯抬脚踢起地上的马镫。铁镫撞上枪口，枪声没有响，子弹打进房梁，木屑雨点一样落下。他顺势抓住枪手衣领，把人拖到油灯前。

油灯火苗猛地一晃。

银柄左轮这才出现在伊莱亚斯手里。枪口没有对准人，而是压住油灯铜座。冰冷的银柄贴着灯沿，火苗像被什么看不见的东西按低了一寸。

“谁点火，谁先烧。”

马车房里的马全安静了。

账房抱着棕色账夹往后退，被乔纳从后门一把拎住。乔纳把他按在车轮上，膝盖顶住他腿弯：“跑错门了。”

埃莉诺看着伊莱亚斯，呼吸仍稳，只是右腕被扣过的地方手套皱了。她没有喊救命，也没有把账夹递给他，而是先低头看自己腕口。

“手套脏了。”她说。

伊莱亚斯看了一眼扣她手腕的枪手。

那人已经被他按在地上，手指还想摸向靴筒里的短刀。伊莱亚斯脚尖一踩，短刀连鞘一起滑出，撞到埃莉诺裙边。

“现在干净了。”

埃莉诺弯腰拾起短刀，用两根戴手套的手指捏着，放到旁边车厢踏板上。动作优雅得像在整理餐刀。

账房开始发抖：“韦德先生，这只是家庭财产纠纷。夫人拿走的是范恩先生账夹，我奉命——”

伊莱亚斯把枪口从油灯上移开，指向地上的火柴盒。

“家庭纠纷带火柴？”

乔纳从账房怀里搜出第二盒火柴、三张空白车厢事故证词和一枚铁路董事会小印。

埃莉诺看见那枚小印，眼神终于变了。她走上前，伸手。

乔纳看向伊莱亚斯。

伊莱亚斯点头。

小印落到埃莉诺掌心。她合拢手指，白手套包住那一点金属，像包住从自己身上割下的旧枷锁。

“这是我的印。”她说，“我父亲死前给我的。范恩说它遗失在东部办公室。”

账房急忙道：“夫人记错了。”

埃莉诺转身，抬手就是一记耳光。

白手套打在脸上的声音清脆，账房被打得偏过头，嘴角渗血。马车房里几个枪手同时僵住，没人想到她会在韦德面前自己动手。

“我记错的东西很多。”她声音很低，“但自己的印不会。”

伊莱亚斯看着账房：“谁让你拿印？”

账房闭嘴。

伊莱亚斯把蓝边杯底蜡片从乔纳手里接过，放到油灯旁。蜡片边缘已经发黑，像有烟从内部透出来。账房一看见那串账号，脸上的血色退得干干净净。

马车房里忽然冷了一点。

地上的枪手抬头，眼神惊恐，像听见草料堆后有马刺声一步步靠近。可那里什么也没有，只有干草轻轻下陷。

“我说。”账房嗓子发哑，“账号是董事会暗户。范恩先生用它付派克森选举费，也付给货栈的人做箱子。夫人发现后，要我把凭证交出来。我本来想给她，可范恩先生派人盯着——”

埃莉诺冷笑：“你想给我，还是想卖第二次？”

账房不敢看她。

伊莱亚斯蹲下，把火柴盒、事故证词、小印和账夹一件件放在车厢踏板上。

“账夹打开。”

埃莉诺没有立刻动。她看着那只棕色账夹，指尖压在锁扣上，白手套皱褶慢慢舒平。

“这是我的。”她说。

“所以你开。”

她眼里的冷终于松开一点。她从项链坠里抽出一把极小的钥匙，插进账夹锁孔。咔哒一声，锁开了。

里面是三张汇兑凭证、一份董事会临时授权副页和一张舞会主桌账单。账单上，蓝边香槟杯编号、主桌座位和伪选票箱货栈费用写在同一行。

乔纳低声骂了一句。

伊莱亚斯把凭证交给克拉拉的报童。报童不知何时从花坛边钻进来，脸上沾着泥，眼睛亮得吓人。

“送侧厅，抄三份。”

报童拔腿就跑。

马车房外有脚步靠近时，埃莉诺先把账夹合上。她没有躲到伊莱亚斯身后，而是把自己的金融小印握在掌心，站到油灯光能照见的位置。

乔纳把三个枪手的枪一支支卸下，排在马车踏板上。每排一支，门外偷看的宾客就低吸一口气。那些枪都不是普通护卫枪，枪柄内侧刻着铁路货栈编号，其中一支编号和伪选票箱夹层票根上的货栈尾号相同。

伊莱亚斯拿起那支枪，丢到账房脚边。

“谁签领？”

账房额头贴着车轮，嘴唇哆嗦：“范恩先生办公室。”

“办公室不会握枪。”

“是……是霍布斯队长。”

乔纳立刻记下名字。

埃莉诺补了一句：“霍布斯只听范恩私人命令，不听董事会。若他领了货栈枪，说明范恩把董事会的钱转成自己的枪。”

门口几个董事会代表脸色变了。一个人低头看自己的袖扣，另一个把原本要替范恩递出的名片收回内袋。

伊莱亚斯看见他们的反应，便把那支枪递给最近的董事代表：“拿好。待会儿回大厅，别说没看见。”

那代表像接到烫铁，拿也不是，丢也不是。

伊莱亚斯又从踏板上拿起补妆房钥匙，抛给埃莉诺。钥匙落进她戴着白手套的掌心，发出一声清脆小响。

“谁的门？”他问。

埃莉诺看着钥匙齿口。那里原本刻着她父亲家族的小纹章，如今被粗糙磨平，又压上范恩办公室号。

“曾经是我的。”

“现在呢？”

她把钥匙穿进账夹锁扣，反手扣住：“现在先开我的账。”

门外偷看的几个东部夫人看见这一幕，扇子遮得更低。她们懂钥匙比枪更难夺回来。范恩能说枪手只是护卫，却不能解释一个丈夫为什么把妻子的补妆房钥匙交到账房和火柴一起藏在马车房。

伊莱亚斯用靴尖把火柴盒踢到油灯另一侧，让所有人看清火离干草只差一掌。他没有大声申斥，只低头看着账房：“再说一次，家庭纠纷？”

账房喉咙滚动，没敢答。

就在这时，门外响起范恩的声音：“韦德先生，您在我的马车房里审我的人？”

他走进来，身后跟着两个州府特派员和几个被惊动的宾客。灯光从门口涌入，照见地上的枪手、火柴盒、事故证词、补妆房钥匙和埃莉诺手里的小印。

范恩的视线停在埃莉诺脸上：“亲爱的，你受惊了。”

埃莉诺站直，账夹抱在身前。

“受惊的是你的账房。”

范恩笑容不改：“你拿了不该拿的东西。”

伊莱亚斯把银柄左轮收回枪套，枪扣合上的声音很轻。

“火柴、假证词、暗户凭证和她的印。”他说，“你说哪件？”

宾客们开始窃窃私语。薇拉站在门外，扇子挡着嘴，眼神兴奋得像看见一场真正的决斗。格蕾丝则死死盯着埃莉诺手腕上被攥皱的手套，脸上第一次没有羞怯，只有愤怒。

范恩终于看向地上的账房。

账房张了张嘴，却在碰到伊莱亚斯目光时缩成一团。

埃莉诺把那只托盘从第三辆马车座下抽出来。托盘银面被烟熏过，边缘压着一张主桌杯号单，杯号单背面写着几行小字：若夫人离席，杯碎；若韦德离席，火起；若账夹离房，证词递交。

她把托盘放到门口，让所有宾客都看见。

“这是他给我的舞会安排。”

薇拉合上扇子，脸上的笑第一次收起：“东部戏院里，反派都比这体面些。”

格蕾丝把被撕裂的裙摆布条递给埃莉诺，低声道：“夫人，包一下手腕。”

埃莉诺接过布条，没有说客套话，只把布条缠在被攥皱的手套外。这个小动作让门外黑河姑娘们的眼神变了。她们原先看见的是铁路夫人，现在看见的是一个被人用火柴和假证词堵在马车房里的女人。

伊莱亚斯把杯号单压到范恩面前：“你安排得很细。”

马车房角落里，油灯又轻轻一响。火苗一瞬间变成冷白，随即恢复。

范恩脸上的笑裂开一丝。

伊莱亚斯拿起那份舞会主桌账单，按到马车房门板上。

“今晚不跳舞了。”他说，“回大厅，读账。”

# 第218章 林月兰压住银星热

伊莱亚斯从旧货场回来时，药铺后门已经留了半掌宽的缝。林月兰站在门内，手里提着一盏罩了蓝布的灯，灯光不亮，却足够照见他右手的烫伤、膝侧的裂口，以及眼底那点不该被外人看见的冷热。

“进来。”她没有问他还能不能走。

伊莱亚斯本想说先去公馆签口供，话到嘴边，被她看了一眼，又咽回去。他跨进后间，门在身后合上，外街的风声和远处报馆的机器声都被隔在木板外。药铺里全是草药、酒精和湿纱布的味道，干净得近乎严厉。

林月兰让他坐到矮床边，先剪开他手套。皮革粘住烫伤的地方，她动作很轻，剪刀却稳。伊莱亚斯额角出了汗，仍笑着说：“我以为你会先骂我。”

“骂要耗气。”她把手套碎片放进铜盘，“你现在不配让我耗。”

他低低笑了一声，笑到一半，指尖忽然一颤。那不是疼痛本身，而是旧货场、旧歌、梅森的空枪和残缺名单一齐压回来后，引出的另一阵热。它从胸口往眼底涌，像有看不见的火沿血脉找出口。外人若看见，只会把它当成怪异，当成能被梅森利用的新把柄。

林月兰比他更早察觉。她立刻把蓝布灯罩压低，又将屏风往床前推了半尺。后窗纸上正贴着一道人影，影子很薄，像有人弯腰听墙。

“阿贝。”林月兰淡声开口，“你再靠近一步，我就把你耳朵里的冻疮药换成泻药。”

窗外影子僵住。片刻后，一个男人讪笑：“林医生，我只是路过。听说韦德先生受伤，街上都关心。”

“关心写在门诊簿上，不贴在后窗。”她把木簪从发间取下，轻轻敲了敲窗框，“去前堂坐。若是梅森让你来数他眨几次眼，你就告诉梅森，病人发热，医嘱不许参观。”

脚步声退远，却没有完全离开。林月兰不再理会，转身用湿纱布覆上伊莱亚斯手背。冷意压下第一层热，他肩膀终于松了半分。

“看我。”她说。

伊莱亚斯抬眼。那点异样的亮意被他强压着，压得眼眶发红。林月兰没有惊慌，也没有追问。她只是伸出两指按住他腕脉，另一手把红线药瓶打开，滴了三滴在纱布上。药味苦而清，像冬夜里剥开的树皮。

“呼吸跟着我。”她放慢声音，“吸气，停，吐。不要去想车顶，也不要去想梅森刚才那句话。你现在只在药铺后间，门锁着，灯遮着，我在这里。”

伊莱亚斯照做。第一次呼吸断在喉咙里，第二次才勉强顺下去。那股热仍不肯退，像被旧事喂了一把干草。林月兰用干净布条绕过他手腕，把纱布固定住，又拿起一件深色外衫披到他肩上，遮住颈侧逐渐浮出的浅痕。

“有人问，就说是车顶煤烟呛的。”她说。

“这不像好谎。”

“医嘱不叫谎，叫病情简述。”

伊莱亚斯看着她，笑意疲惫却真了一点。“你学坏了。”

“我学会不给敌人开药方。”林月兰把空药瓶收进抽屉最里层，“梅森想知道你哪里会疼、什么时候会失控、谁能让你停下来。今晚他已经派了三个探子，一个在前堂买止咳糖，一个在后窗听墙，还有一个在街角盯灯色。我不会把病人送给他们看。”

她说得平静，手上动作却没有停。清创、包扎、换水、烧掉带血纱布，每一步都避开外人的眼。连铜盘里的水，她也倒进药渣桶里，用苦艾叶盖住颜色，再把药渣桶推到内门后，离后窗最远。

外间传来争执声。阿贝果然不肯走，正和药铺学徒磨牙，说报馆都能写韦德旧账，药铺凭什么不让人探病。林月兰把帘子掀开一线，声音不高：“报馆写证据，药铺治病。你若分不清，我可以先治你的嘴。”

学徒噗地笑出声，阿贝骂骂咧咧离开。林月兰落下帘子，回到后间时，伊莱亚斯已经把那阵热压下大半。他额发湿着，脸色白得厉害，却没有躺倒，也没有用玩笑把一切糊过去。

“你可以承认疼。”林月兰说。

“疼。”他答得很快。

这反而让她停了一下。伊莱亚斯看出她的意外，靠在墙上轻声道：“旧桥、火车、梅森，哪一件都够疼。疼不是麻烦。麻烦是他们想让我疼到认错账、开错枪，把今天的锅扣到昨天死人头上。”

林月兰把药碗递给他。“那就先喝药。锅再重，也得有手端稳。”

药苦得近乎粗暴。伊莱亚斯喝完，眉头都没皱，只问：“公馆那边？”

“露丝派人来过。口供能等半个时辰。克拉拉在写车厢弹道，乔纳在审断耳信使，娜奥米送两个孩子回家。没有一件要你现在爬起来证明自己还站得住。”

“听起来我很没用。”

“听起来你终于有同伴。”林月兰把药碗收走，语气淡淡，“这不丢人。”

后窗外又有轻响。林月兰这次没有出声，而是把一只空药箱塞到伊莱亚斯手边，自己吹灭蓝灯，只留前堂灯光透过帘缝。影子贴近纸窗的瞬间，伊莱亚斯抬手掷出药箱。木箱砸开窗闩，正中探子的鼻梁。阿贝惨叫一声摔进后巷，乔纳安排的巡夜人立刻把他按住。

林月兰重新点灯，瞥了伊莱亚斯一眼。“手不能用，你还扔？”

“左手。”

“左手也在医嘱范围内。”

他识趣地闭嘴。

阿贝被拖到前堂时，身上搜出一张小纸，纸上写着：看眼、记热、查药。林月兰把纸放进证物袋，却没有让任何人进后间。她只在门诊簿上写下公开病名：煤烟灼伤、旧创发热、需静养。字迹端正，足够应付街上的好奇，也足够挡住梅森想要的细节。

伊莱亚斯看着那行字，低声问：“你不怕将来有人说你替我遮掩？”

“我是在替病人保密。”林月兰合上簿子，“若你杀了人、偷了证物、烧了桥，我不会替你藏。可你现在只是伤病发作，而敌人想把它当武器。我不欠他们病历。”

伊莱亚斯沉默片刻，看着门诊簿上那行端正字迹，点了点头。

半个时辰后，他终于能站起来。林月兰给他换了一副宽松手套，外面看不出烫伤，也不会牵扯纱布。她又把他的帽檐压低一点，遮住眼底残余的红。

“去公馆可以。”她说，“不许拔枪，不许喝酒，不许逞强到倒在街上。”

“第三条有点难。”

“那我让乔纳把你绑回床上。”

伊莱亚斯笑了，推门前却认真说了句谢谢。林月兰没有接这声谢，只把药箱递给随行的巡夜人，叮嘱他每隔一刻看一次手套边缘。后巷风冷，街角还有人探头探脑，但他们看见的只是一个受了车顶烫伤仍能去签口供的男人，看不见药铺后间那场无声的照料与遮掩。

梅森想要的细节，被一盏蓝布灯、一张门诊簿和林月兰冷静的手挡在了窗外。

等伊莱亚斯离开，林月兰没有马上熄灯。她把烧掉的纱布灰烬压进药渣底层，又把门诊簿翻到前一页，检查自己写下的每个字是否足够平常。煤烟灼伤，旧创发热，需静养。没有多一笔，也没有少一笔。遮掩不是编造另一个离奇故事，而是把病人的隐私留在病人该在的位置。

前堂学徒端着水进来，小声问：“林医生，他是不是病得很重？”

林月兰接过水盆。“重不重，都不是你在柜台上讲给客人听的事。”

学徒立刻点头，又忍不住说：“可梅森的人一直问。他们说若我们藏着不说，药铺就是帮韦德欺骗全城。”

林月兰把水倒进炉边，热气很快遮住铜盆里的淡色。“全城该知道他有没有杀人、有没有偷证物、有没有烧桥。不该知道他的脉什么时候乱、哪味药能压住、哪扇窗最暗。把这些说出去，不是诚实，是给敌人递刀。”

她说完，自己也静了一会儿。随后她把门闩落下，把灯罩往低处拨，又倒了一杯温水放到伊莱亚斯手边。杯沿离他右手只有半寸，他不用抬眼也能碰到。

后巷传来巡夜人押走阿贝的脚步声。阿贝还在喊，说自己只是想写一份卖给梅森的病情。林月兰推开窗，声音清冷：“告诉梅森，若他想买药方，先挂号；若他想买病人，我这里不卖人。”

窗外安静了。她关窗，重新把木簪插回发间。那支木簪在今天挑灭过灯、敲退过探子、压住过门诊簿，像一件很小的兵器。林月兰知道，接下来的日子里，她守的未必是战线中最响的一条，却一定是梅森最想偷看的一条。她不会让他看见。

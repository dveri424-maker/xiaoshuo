# 第128章 电报局第一次听警署

旧货棚二号轨堵得正好。

一列空煤车被莉迪亚调到岔道上，车厢横在旧货棚门口，像一堵黑铁墙。十几个私兵刚从棚里冲出来，就被煤车挡住退路。乔纳带退伍兵从左侧压上，玛丽安的马队从右侧封街。

伊莱亚斯从正面走来。

站台灯火照着他黑发灰眼，银星徽章在夜色里冷亮。私兵领头人拔枪，银光还没在旁人眼里出现，枪已经被伊莱亚斯一枪打飞。第二人想翻煤车，被礼拜日冲到车边，一口咬住他外套后摆，硬把人拖下来。

“好马。”玛丽安在马背上吹了声口哨。

伊莱亚斯笑了一下：“它记仇。”

枪战没有拖长。

私兵被堵在煤车和货棚之间，脚下全是散煤，跑一步滑半步。乔纳的退伍兵用木棍敲膝弯，玛丽安的骑手用套索拉枪手，伊莱亚斯只开了三枪，三枪都打枪带、灯绳和门锁。等车站钟敲过九下，十六名临时枪手全部趴在煤灰里。

克拉拉赶到时，正好看见最后一个人被按到煤车轮边。

“别动。”乔纳说，“煤车不挑脚。”

莉迪亚的电报纸一张接一张送来。每张都用不同假词开头，却指向同一件事：药铺街北口还有三人，华工营地后水沟两人，黑学校侧巷四人，报馆窗下一个递火油瓶的孩子。

伊莱亚斯站在站台桌前，把电报纸一张张按到地图上。

“从现在起，电报局接警署调令。”

车站调度员脸色发白。

“电报局归铁路调度——”

银星徽章按在地图中央。

“今晚归黑河。”

克拉拉抬笔：“这句我能写吗？”

“写。”

玛丽安把铜哨吹响，马队分出两骑去药铺街。乔纳带退伍兵去黑学校侧巷。露丝派来的旅店伙计拎着灯，接过电报纸往公馆跑。整个黑河镇像第一次用同一根线呼吸。

电报局里，莉迪亚坐在桌前，左手边是小铁盒，右手边是黄铜电键。她的纱布指尖敲得慢，却稳。每一条消息进来，她先标时间，再标地点，最后用红铅笔写“警署已接”或“马队已去”。

萨姆趴在窗口问：“格兰特小姐，车站那边问，为什么二号轨堵着？”

莉迪亚眼睛不抬。

“回他们，煤车迷路。”

萨姆咧嘴跑了。

药铺街北口的三人被林月兰和华工们堵住。林月兰没有动刀，只把蓝砂粉撒到他们袖口，袖口立刻泛黑。华工们一看，木棍齐齐抬起。三人跪得很快。

黑学校侧巷的四人被娜奥米的门队抓住。乔纳赶到时，本杰明已经用独臂把一个人按在水桶旁。娜奥米站在门前，教鞭点着地上的火油罐。

“名字。”

那人不说。

“全名。”

乔纳笑：“你最好说，老师比牢房难熬。”

报馆窗下的孩子被克拉拉亲自逮住。那孩子手里拿着火油瓶，哭着说是铁路给钱。克拉拉没有骂他，只把火油瓶放到桌上，让他坐在印刷机旁看号外一张张吐出来。

标题：电报局第一次听警署。

半夜前，所有电报纸、臂章、火油瓶、蓝砂袋、假调度和私兵名单都送进警署。莉迪亚也来了。

她坚持自己走来，手上缠着纱布，脸色有点白，怀里抱着小铁盒。伊莱亚斯站在警署长桌边，正在看乔纳押进来的最后一批人。

“原译本。”莉迪亚把铁盒放到桌上，“按时间顺序。副本给报馆，简码给电报局留档。还有我自己写的调度说明，防止明天有人说煤车自己迷路。”

乔纳笑出声。

伊莱亚斯打开铁盒看了一眼，又合上。

“钥匙。”

莉迪亚把一把小钥匙放到桌上，另一把自己握着。

“这把给警署，不给铁路。”

“电报局呢？”

她抬起下巴。

“电报局今晚已经听过警署了。以后铁路要调黑河线，先把明码送一份到警署桌上。暗码我来拆。”

车站调度员在旁边脸色发苦。

“这不合规矩。”

莉迪亚看他。

“今晚你的规矩差点把我烧死。”

调度员闭嘴。

伊莱亚斯把银星徽章按到铁盒盖上，留下淡淡灰痕。

“黑河电报局临时密码室。莉迪亚·格兰特管原译本，警署管调令副本，报馆管公开副本。谁撬盒子，按撬警署证物。”

莉迪亚的眼睛亮得厉害。

她努力把笑压下去。

“我需要一张桌。”

露丝正好端咖啡进来。

“公馆东侧还有半张。别烧。”

克拉拉说：“报馆也能给你一角。”

娜奥米站在门口，听见后说：“学校可以给你夜课黑板。”

莉迪亚看着这些女人，忽然不知道该先看谁。她抱紧铁盒，年轻的脸上第一次没有急着证明自己，只是认真点头。

“我会用上。”

警署外，夜车汽笛低低响了一声。车站那边煤车还横在二号轨，铁路的人围着它骂，没人敢私自挪。

伊莱亚斯走到门口，看着黑河镇的电线从街顶穿过。风吹过线，发出很轻的嗡鸣。

莉迪亚站到他身后半步。

“听见了吗？”她问。

“什么？”

“线在响。”

伊莱亚斯懒散一笑。

“让它以后响给我们听。”

远处药铺街忽然亮起灯。一个华工少年跑来，喘得说不出整句。

“林医师……药铺门口……有人死了！”

莉迪亚怀里的铁盒一沉。



莉迪亚把铁盒交给露丝后，又立刻要拿回来。

露丝没有笑她，只把两把钥匙分开。一把挂在莉迪亚颈上的细绳里，另一把扣进自己钥匙串最里面。钥匙串一响，莉迪亚看见那上面已经有公馆内门、储物间、东柜、档案副箱的钥匙。

“我的也挂那里？”

“挂得住。”露丝说。

“若铁路来要？”

“让他先报真名，再让他等。”

莉迪亚低头看自己那把钥匙，忽然笑了一下。

娜奥米走过来，把一张夜课黑板使用表递给她。

“你若要教孩子识电码，先写课程名。”

“他们会学吗？”莉迪亚问。

“他们已经学会银行会骗人，电码不算最难。”

克拉拉把报馆一角的桌牌也递给她。

“报馆给你半张桌，条件是暗码标题归我先挑。”

玛丽安从门口经过，丢来一只小铜哨。

“马队传信哨。别乱吹，除非真要马。”

莉迪亚手里忽然多出铁盒钥匙、黑板表、报馆桌牌和铜哨。她抱得有些狼狈，却舍不得放下任何一样。

伊莱亚斯看着她。

“现在够桌子了？”

莉迪亚抬头，眼睛亮得像电火。

“还缺一张线路总图。”

他笑了。

“自己画。”

她点头，把所有东西放进铁盒旁的小布袋。警署桌上的银星徽章灰痕还在，像给她这半夜的莽撞和本事都盖了章。

伊莱亚斯转身，银星徽章在警署灯下冷亮。

华工少年喊完，警署里刚松开的气又绷紧。

林月兰已经在门外。她方才送药粉到警署，还没来得及坐下，听见“药铺门口有人死”，药箱搭扣立刻扣上。伊莱亚斯看她，她没有解释，只把红线药瓶往腰侧一按。

“我先去。”

“不。”伊莱亚斯说，“一起。”

莉迪亚抱着铁盒站起来。

“我能听药铺街线。”

“药铺没有电报线。”乔纳说。

“但车站有，矿区有，铁路的人若要栽给林医师，会向外报。”她把铁盒交给露丝，又坐回临时桌前，“给我一根接到车站的明线。”

露丝立刻看向霍布。

霍布扛着工具箱进门：“我只是铁匠，不是电线匠。”

莉迪亚抬头：“铜丝会不会接？”

“会。”

“那你就是半个。”

霍布骂了一句，还是跟她去门口拉线。克拉拉拿起本子，娜奥米带门队跟上，玛丽安的铜哨已经在街外响起。

伊莱亚斯翻身上马，药铺街的灯火一盏盏亮起来，像有人在夜里把一串证物摊开。

霍布接线时，莉迪亚就坐在警署门槛上。

她把黄铜耳机扣在一只耳朵上，另一只耳朵听街上的动静。铜丝从警署窗框绕出去，搭到车站旧线上，中间用霍布的铁夹夹住。每夹一下，线就轻轻颤一下。

“这样会不会炸？”霍布问。

莉迪亚抬头：“你接铁轨时会不会问铁会不会生气？”

霍布哼了一声，继续夹。

第一串声音进来时，她立刻抬手。

“车站在报药铺街。有人说‘死人已到门口’。”

伊莱亚斯的手已经落到马鞍上。

莉迪亚继续听，脸色更冷。

“还有一句：林若不认，封华工营。”

林月兰的名字没有完整出现，却已经被暗码咬住。警署门口一静，几道目光同时落到她腰侧的药瓶上。

克拉拉把这两句写在纸上，递给林月兰。

林月兰看完，折好，塞进药箱侧袋。

“他们替我写罪，我替他们留方子。”

莉迪亚把耳机压紧。

“还有新码。白医动身。”

伊莱亚斯翻身上马。

“那就去见白医。”

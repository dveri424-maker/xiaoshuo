# 第411章 城邦两个字先被骂

市政厅门口的传票被雨水泡了一夜，红蜡却还硬得像一块凝血。

三日后公开审查黑河城邦宪章。

这行字贴在门板中央，旁边还钉着州府蓝边副本。清晨第一拨矿工路过，没敢伸手撕，只围着看。学校街的孩子被娜奥米带走，回头张望时，看见东部律师的马车已经停在台阶下。

马车门开，先落下一只擦得发亮的黑皮鞋。

阿尔杰·昆西，东部来的宪法律师，领结打得细，手杖银头刻着一截树枝。他身后跟着州府副秘书官马洛里，灰大衣下摆不沾尘，抱着一只装满文件的皮箱。

黑门那边也来了人。老黑门头目费根把帽檐压得很低，两个赌场账房抬着一只小铜柜，铜柜上还留着昨夜酒渍。商会会长惠特曼站在另一边，手里捧着一本厚账，脸上挂着等价码的笑。

布莱斯法官站在市政厅门内，木槌还没敲，厅里已经分出两边。白天穿大衣的人坐靠窗，夜里靠赌桌吃饭的人站在墙根。中间那张长桌空着，桌面擦得太干净，反倒像等一场枪战。

门外马蹄声停下。

伊莱亚斯·韦德走上台阶，黑大衣被风掀开一角，银柄左轮沉在腰侧。乔纳带银星骑警列在门外，露丝抱着黑门账夹，克拉拉夹着铅笔和空白版样，埃莉诺的白手套按在基金章程上。

他没有看传票。

他抬手，把那张纸连同钉子一并拔下。钉尖带出一小片门漆，落在掌心。

“韦德先生。”昆西律师起身，声音很亮，“我代表东部债权人、州府备案委员会以及若干受影响商户，对所谓黑河城邦宪章提出异议。美国土地上没有私人城邦。”

厅里有人低笑。黑门的人笑得难听，商会的人笑得小心，州府书记员飞快蘸墨。

伊莱亚斯走到长桌前，把传票反扣在桌上。

“坐。”他说。

昆西没有坐，反而把手杖银头往桌沿一碰：“我先请您解释两个字——城邦。这个词是谁给您的胆子？”

墙根的费根吹了声口哨：“我也想听。白天的市长要管夜里的门，总得让夜里的人分把刀。”

惠特曼会长笑着合上账本：“商会只关心账。若要发行城邦债券，商会必须有席。”

马洛里副秘书官咳了一声：“州府关心边界。黑河仍在州府辖下，不可自行铸章、设武装、征收特别款。”

四边的声音压到一张桌上。

布莱斯抬槌，却没敲。他看向伊莱亚斯。

伊莱亚斯把手伸向露丝。

露丝上前，先放下黑门钥匙。旧铜钥匙砸在桌面，响声浑厚，黑门那边几个人不笑了。

乔纳进门，递上旧城徽。那枚被法官助手偷过的城徽边缘有缺口，缺口旁还留着封蜡。伊莱亚斯把它压在传票左上角。

露丝又打开盒子，取出黑河市长印。印章沉沉落在旧城徽旁。

埃莉诺摊开国会议员副本，白纸红印，纸角压住昆西律师递来的异议书。

最后，伊莱亚斯从腰间取下黑门钥匙，放在市长印另一侧。

旧城徽，市长印，国会副本，黑门钥匙。

四件东西把那张长桌压得安静下来。

伊莱亚斯抬眼：“谁不认，先说哪件不算。”

昆西律师手杖停住。

州府副秘书官低头翻文书，翻得太急，纸页刮破了手套。商会会长的笑僵在嘴角。费根盯着那把黑门钥匙，眼皮抽了一下。

克拉拉的铅笔落下第一行字，声音比木槌还清楚。

昆西缓过来，仍旧体面：“旧城徽只证明历史遗留。市长印只管市政。国会副本不赋予您分裂州权。至于黑门钥匙——那更是非法地下秩序的污点。”

艾达站在门边，冷笑一声：“污点给医院买了床，给学校买了灯。你那只干净手套买过什么？”

昆西没理她，看向布莱斯：“法官，请记录。你们把市政厅的章、韦德基金的柜、黑门的赌税和银星骑警的枪全堆到一张桌上，还敢给它换个好听名字。”

伊莱亚斯指节敲了敲黑门钥匙。

“费根。”

老头目抬头。

“黑门昨夜收了多少夜线劳务票？”

费根不情愿地看向露丝。露丝已经翻开账夹，冷冷念出数字，又把一串票根推到桌上。票根上沾着酒味、煤灰和红印，底下夹着医院床号、学校灯油票、矿工马料账。

伊莱亚斯拿起最上面一张，扔给昆西：“念。”

昆西皱眉：“我不是账房。”

“你刚才骂它非法。”伊莱亚斯靠近半步，“先把它花在哪儿念出来。”

昆西没接，票根落在他脚边。

一名矿工家属从旁听席起身，袖口还别着医院床牌。她弯腰捡起票根，嗓子发哑：“三十七号病床，换药纱布，夜税柜支出。”

厅里静了。

又一个学校街家长拿起第二张：“灯油两桶，门队夜课。”

第三张被乔纳拾起：“银星骑警马料，护送毒粉课本证物。”

伊莱亚斯没有解释。他只是让一张张票根从白天的人手里传到夜里的人手里。赌桌钱、学校灯、医院床、骑警马料在桌面上排开，每一张都带着手印和日期。

昆西的脸色沉了一点。

马洛里副秘书官终于开口：“即便如此，宪章二字仍须州府审阅。州府不能允许黑河绕过既有章程。”

布莱斯敲下木槌：“今日就是公开审查。”

“审查可以。”伊莱亚斯把传票翻正，钉孔裂在红蜡边，“审查的人也得留下东西。”

他看向商会会长。

惠特曼立刻笑：“商会愿意配合。只要未来市政债券、城邦债券、基金支出，商会有一票否决权。没有商人的信用，黑河印再多纸也没人收。”

黑门墙根传来低低骂声。

费根眯眼：“白手套想坐第二把椅子？那夜路要不要也给你们盖章？”

惠特曼不看他，只看伊莱亚斯：“市长先生，我们不反对您写宪章。我们只要求把商会否决权写进去。”

他说着，把厚账本推到桌边。账本扣带镶银，像一把不露刃的刀。

伊莱亚斯拿起那本账，没翻。

他把账本放到市长印、旧城徽、国会副本和黑门钥匙旁边。

“好。”他说。

惠特曼眼里刚露出喜色。

伊莱亚斯接着说：“明天带你们的否决权来。先从医院欠账念起。”

惠特曼的笑停住。

克拉拉抬头，铅笔尖悬着：“标题我有了。”

布莱斯木槌再落一声，审查开场的签名簿被推到桌中央。昆西律师第一个签，笔锋硬得划破纸背。马洛里副秘书官跟上。惠特曼签名时手腕僵硬。费根最后按了个黑墨手印，按完还用袖子擦了擦，像怕白纸咬人。

伊莱亚斯最后拿起笔，没有签在他们后面。

马洛里副秘书官想把旧城徽往自己那边挪半寸。乔纳没动枪，只把手掌按在桌沿。

布莱斯看见了，木槌轻轻一敲：“见证物不得离桌。”

马洛里收回手，指尖沾了一点旧铜绿。他掏手帕擦，擦得太用力，手帕边缘被染出一抹暗绿。旁听席里一个老车夫低声笑：“州府手也会脏啊。”

昆西回头瞪去，老车夫却把裂轴账单举起来，半点不躲。那张账单曾在市政台阶上压过国会议员文书，如今又被他带来，像一块不肯离场的旧木刺。



他把黑河城邦宪章草稿封面放到签名簿上方，用旧城徽压住封角。

“审。”他说，“但桌子在黑河。”

门外风把传票留下的门漆吹进厅里。一枚小木屑落在宪章封面上，克拉拉看见，伸手没有拂掉。

那是市政厅门上被拔下的钉痕。

惠特曼会长抱起自己的账本，临出门前低声说：“明天商会带账来。没有否决权，就没有债券。”

伊莱亚斯看着他下台阶。

黑门钥匙在桌上冷冷一响，像夜里有人把门闩推上了。

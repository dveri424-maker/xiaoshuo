# 第222章 露丝锁住第三道门

公馆有三道门，这件事以前只有住在里面的人在意。

第一道是临街外门，门板厚，挂着铜铃，镇民交口信、送账单、避雨都从那里进。第二道在前厅后面，通向证物室和露丝的小书房，钥匙只给登记簿认。第三道最窄，藏在楼梯阴影里，后面是内宅、病床、孩子曾经躲过枪声的壁炉，也是梅森的人最想摸进去的地方。

黑钟挂上门柱后的第二天，露丝把三道门的钥匙摆在长桌上。她没有穿最华丽的裙子，只系了一条灰色腰带，钥匙从腰带边垂下来，碰一下就响。外门排着交证物的人，二门前坐着克拉拉，第三道门前没有人，只有一把椅子和露丝本人。

伊莱亚斯站在前厅窗边，银柄左轮没有解扣。他看露丝检查门闩，像看一名老军士检查火药包。

“梅森会派人从熟人里进来。”他说。

“所以熟人也按陌生人的规矩来。”露丝答。

第一批来的正是熟人。给公馆送过十年牛奶的老汤姆，怀里抱着一只空桶，说是艾达让他送清水给女工。他低着头，脚步比平时快半拍。露丝请他把桶放在外门线前，桶底一翻，夹层里滚出一枚短钉和一片封蜡，封蜡上印着梅森账房的月桂纹。

老汤姆脸涨得通红，差点跪下。他说儿子欠了赌债，赌场的人让他把封蜡藏进第三道门里，夜里自然有人来取。露丝没有骂他，只让克拉拉记下赌债数目和威胁原话，再让女仆给他一杯热咖啡。

“你今天只能进第一道门。”露丝说，“你儿子的账，明天到公馆外门公开算。”

这比赶他走更重。老汤姆抱着空桶坐到外门边，像被一条看不见的线钉住。围观的人窃窃私语，梅森的眼线却没法把他变成英雄或叛徒；他只是一个被买过、又被登记下来的证人。

第二个试门的是穿黑裙的寡妇。她说要给楼上伤员送药，药包里确实有草药，也有一截细铜丝。林月兰从后廊出来，指尖一捻便闻出草药被赌场仓库的灯油熏过。她没有多话，把真正能用的药留下，铜丝交给克拉拉，黑裙寡妇交给外门排队的退伍兵看住。

“药可以进第二道门，人不行。”露丝说。

第三次试门来得最巧。午后风大，门口黑钟被吹得乱晃，一个戴毡帽的年轻人趁人群抬头时从侧廊溜进来。他手里没有枪，袖口干净，脸上还带着送信人的怯意。可他走向的不是证物室，而是第三道门，步子每一步都踩在木板不响的位置。

伊莱亚斯没有追。他只是把窗边的帘绳往下一拉，前厅光线变暗，第三道门前那道细灰线却亮出来。年轻人的靴尖正好踩在线内半寸。

露丝从椅子上站起，钥匙串在掌心一合。

“你过线了。”

年轻人笑得勉强：“夫人，我只是迷路。”

“公馆里会迷路的人，先问门，不先找锁。”

他终于想拔袖里的小刀。刀柄刚露出来，伊莱亚斯的银柄左轮已经指在他影子上，而不是胸口上。那位置逼得他若想前扑，必须先把自己的脚从灰线里拔出来。露丝抬手，女仆从两侧合门，门闩咔哒一声，将第三道门彻底封死。

年轻人的肩膀垮下去。他袖里除了小刀，还有一块薄铜片，边缘磨得极细，正好能试旧锁。铜片背面刻着三个小点，对应黑钟三响。克拉拉把它压在白纸上描图，林月兰检查刀刃，确认上面涂的不是致命毒，只是让人短时发麻的药粉。

“他们不要杀人。”林月兰低声说，“他们要开门。”

露丝看向伊莱亚斯。那一刻，前厅里的人才意识到，公馆真正被争夺的不是某个房间，而是谁有权决定一扇门向谁打开。

伊莱亚斯把枪收回套里，走到三把钥匙前。他没有拿最大的那把，也没有拿最亮的那把，只把代表第三道门的细钥匙推回露丝面前。

“这扇门的规矩归你。”他说。

人群一阵低声。梅森的人最会把女人说成韦德的装饰，说她们只是站在他身后拿着小物件。可今天所有人看见，伊莱亚斯没有替露丝开门，也没有替她锁门。他承认她的钥匙，她就有了公开的权力。

傍晚，克拉拉的号外贴在黑钟旁：三次试门，三次登记，赌场筹码、铜丝、薄铜片一并封存。老汤姆的儿子被乔纳从赌场后门带出来，欠条没有烧掉，而是钉在外门账板上，等明日公算。

露丝最后检查第三道门。门后很安静，壁炉冷着，楼梯上没有陌生脚印。她把钥匙贴进胸口暗袋，转身时看见伊莱亚斯仍站在前厅灯下。

“你不怕我有一天不开门？”她问。

伊莱亚斯笑了笑：“怕。但怕不是理由。”

入夜后，第三道门又被试了一次。这次不是陌生人，而是公馆里最老的厨娘。她端着一锅汤，手抖得厉害，说楼上伤员该喝热的。露丝看着她苍白的脸，没有立刻揭穿，只让她把汤放在第二道门前。林月兰用银针探过汤面，针尖没有变色，锅底却贴着一小片纸，纸上画着第三道门门闩的形状。厨娘当场哭出来，说赌场的人扣了她侄子的赎身契，只要她把门闩样子带出去，明天就放人。

这件事比刀子更难处理。若按旧帮派规矩，厨娘已经算内鬼，拖出去打断手也没人敢说话。露丝却把赎身契的细节问清，交给克拉拉登记，再请乔纳带两名训练队去赌场查契。她没有让厨娘越过第三道门，也没有把她赶出公馆，只把厨房钥匙暂时收回，换给她一盏外门灯，让她在前厅给排队的人倒咖啡。

“规矩若只会惩罚被勒住脖子的人，就和梅森的绳子没区别。”露丝对伊莱亚斯说。

伊莱亚斯听完，第一次认真看那扇窄门。公馆的第三道门守的不只是内宅，也是每个被威胁的人最后一点体面。若所有犯错者都被一脚踢进黑暗，梅森便永远有人可买。露丝把人留在灯下，让错误有名字、有缘由、有后续，反而断了赌场继续操纵的线。

午夜，乔纳带回赎身契。契纸被钉在外门账板上，厨娘亲手按印，承认被逼试门。第二天镇民议论最多的，不是厨娘背叛，而是公馆为何没有私刑。第三道门仍旧紧锁，可门前多了一条新规：被胁迫者先登记，重复隐瞒者再治罪。露丝把规矩写完，钥匙串轻轻一响，像给黑河镇又扣上一枚看不见的锁。

清晨，厨娘照旧煮咖啡，只是站在外门线内给排队的人递杯子。有人避开她的眼神，也有人小声道谢。露丝没有替她求情，更没有让她装作没事，只让她把每一杯咖啡递出去时都说一句：“我昨天被赌场逼着试门，今天在这里登记。”说到第七遍时，厨娘声音不抖了。围观的人也渐渐明白，公馆要的不是一尘不染的人，而是犯错后不能再被黑暗牵走的人。第三道门因此比昨夜更稳，因为它守住的不只是木闩，还守住了人被迫弯下腰后重新站直的路。

第三道门的灯也因此亮到天明。

露丝没有回应，只把外门灯调亮半寸。黑钟在夜风里晃，三道门却各守其位。梅森想用一口钟逼公馆交出心脏，露丝却让全镇先学会了一件事：门不是墙，门是规矩；钥匙在谁手里，谁就能让恐惧停在门外。

# 新版正文前二十章合规复盘与防偏校准

## 文档定位

本文用于复盘 `新版正文/` 第 001-020 章对新版设定的执行情况，并校准第 021-032 章后续写作方向。

本文不是新的章节细纲，不覆盖 `第一卷章节细纲.md`。如本文与 `第一卷章节细纲.md` 在章节收益、关键物件、出场人物或章末钩子上发生冲突，以 `第一卷章节细纲.md` 为准；如确需改变章节收益，必须同步修改 `第一卷章节细纲.md`。

## 前二十章合规结论

第 001-020 章总体符合新版设定硬边界：

1. 章节标题和文件名保持中文。
2. 正文没有提前放入黑河、灰星银体系。
3. 重要人物、地点、机构、矿名第一次出现时，已按需要补充英文原名括注；后续不得重复括注。
4. 伊莱亚斯·韦德始终是场面最终决断者，历史人物和女性角色不能替他定局。
5. 银星眼没有公开解释为系统或可被旁人认知的能力。
6. 艾达、克拉拉、莉迪亚等女性角色绑定工资票、报纸、电报等具体资源，当前不需要削弱。

主要风险不在硬禁区，而在第 017-020 章的县权压力表现：

1. 贝汉县权连续被拆，容易显得过于纸面。
2. 文书、签名、县印、名单、铅字使用较密，后续需要更多牢门、囚车、铁链、听证席位、马队和街面堵截等硬物件。
3. 第 020 章已经得到县印副本铜模、私账、枪手名单、备用链消息和酒馆承诺，这些只能算局部控制，不能当成县权战终局。
4. 怀亚特·厄普只能作为外来强者和观察对象入场，不能被写成贝汉雇佣枪手或脏活成员。

## 第 020 章校准要求

第 020 章中，枪手名单和外来人物观察纸必须在物件层面分开：

1. 枪手名单可以有价码、用途和脏活标记。
2. 外来人物观察纸不得有价码、用途、雇佣标记。
3. 怀亚特·厄普只能出现在外来人物观察纸上，标记为观察，不得出现在枪手名单正文中。
4. 第 020 章保留县印副本铜模、私账、枪手名单、备用链消息和酒馆承诺，但这些只代表韦德抓住贝汉线的局部证据，不代表贝汉县权被彻底打垮。

## 第 021-024 章防偏要求

第 021-024 章的核心任务，是完成“贝汉县权入场并被反用”的八章段落。后续不能继续只用文书打脸，必须出现一次真实县权压迫场面。

必须满足：

1. 贝汉一方要短暂捏住一个公开程序优势，优先在以下三类中选择一类作为主场面：听证席位、继续扣押依据、备用链签收痕迹。
2. 这次优势必须落在可见物件或场地上，例如牢门、囚车、备用链、听证桌、证人座位、县车、后门钥匙或马队堵街。
3. 韦德可以反用县权，但反用结果只能是保住犯人、保住核心证据、拿到继续调查或公开听证位置，不能提前拿到监牢钥匙正式归属。
4. 第 024 章可拿到钥匙去向证据、备用链签收痕迹、听证座位或继续扣押依据；不得写成监牢钥匙正式归韦德。
5. 贝汉不能只输在纸面程序上，必须一度控制门、车、席位、证人或备用链之一，让读者看见县权确实压到了墓碑镇街面。

建议回扣：

1. 玛丽安与圣佩德罗河牧场线应在第 021-024 章回到场面中，可通过马印、偷马路线、囚车路线、牧场证人或水线消息回扣。
2. 露丝的旅店后门资源可承担证人转移、藏人、后门钥匙或听证前夜保护，不应只做早期帮忙。
3. 克拉拉、艾达、莉迪亚可以继续发力，但每次必须绑定自己的资源物件，不得排队送材料。

## 第 025-032 章防偏要求

第 025-032 章的核心任务，是让厄普和霍利迪入场，并在体面展示强度后让位给韦德。

人物边界：

1. 怀亚特·厄普强在枪手判断、街面威望和旧传说压力。他可以识破假枪手、看出街面站位、让镇民短暂转移目光，但不能替韦德完成最终扣押或证据定局。
2. 维吉尔·厄普强在程序漏洞和法律秩序。他可以指出韦德证据链中的程序缺口，但补上缺口的人必须是韦德。
3. 霍利迪医生强在赌桌、枪手圈和近距离压迫。他可以试探韦德，也可以给枪手圈评价，但不能成为替韦德清场的工具人。
4. 厄普、维吉尔、霍利迪均不得成为贝汉脏活成员，不得接受贝汉雇佣，不得参与吓报馆、堵矿工、护县车等脏活。
5. 三人可以制造压力、指出问题、给见证或承认结果；最终证据归属、扣押决定、枪手处置必须由韦德完成。

## 银星眼使用边界

后续应降低完整残影频率。

允许：

1. 通过冷意、银线、纸边发冷、物件异常等短提示，辅助韦德判断方向。
2. 在关键物件上给韦德提供局部判断。

禁止：

1. 让银星眼公开解释为系统、技能或可被他人识别的能力。
2. 让旁人直接理解银星眼机制。
3. 连续用完整残影代替调查、证词、签字、物件归属和街面压迫。

所有公开胜负必须落在证据、物件、签字、证人、报纸、枪械、钥匙、牢门、席位或街面站位上。

## 英文原名括注执行

正文中重要人物、地点、建筑、机构、矿名第一次出现时，可以在中文名后用括号补英文原名。执行边界：

1. 只在首次重要出现时括注。
2. 后续章节只用中文名。
3. 章节标题和文件名不得加入英文。
4. 如后续新增重要历史人物、地点、机构、矿名，应在第一次出现时补充英文原名；不重要的路人和临时物件不必强行括注。

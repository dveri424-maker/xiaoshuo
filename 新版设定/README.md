# 新版设定目录说明

本目录保存新版墓碑镇历史爽文的正式设定文档。

## 阅读顺序

1. `新旧设定融合总则.md`
2. `墓碑镇历史爽文新设定.md`
3. `新版全书卷纲.md`
4. `第一卷章节细纲.md`
5. `新版正文前二十章合规复盘与防偏校准.md`
6. `新版正文主角核心视角与读感校准.md`
7. `旧版资产回收表.md`

## 权威顺序

设定冲突时按以下顺序执行：

1. `新旧设定融合总则.md`
2. `墓碑镇历史爽文新设定.md`
3. `新版全书卷纲.md`
4. `第一卷章节细纲.md`
5. `新版正文前二十章合规复盘与防偏校准.md`
6. `新版正文主角核心视角与读感校准.md`
7. `旧版资产回收表.md`
8. 旧版根目录设定、大纲与正文

`新版正文前二十章合规复盘与防偏校准.md` 只约束执行方向，不新增或覆盖 `第一卷章节细纲.md` 的章节收益、关键物件、出场人物和章末钩子；如确需改变章节收益，必须同步修改 `第一卷章节细纲.md`。

`新版正文主角核心视角与读感校准.md` 只负责正文入口、主角视角锚点、环境描写和读感改稿手法，不新增设定边界，不改变章节收益、人物关系和时间线。合规复盘负责边界，主角核心视角校准负责手法；如发生冲突，以总则、新设定、卷纲、章节细纲和合规复盘为准。

旧版正文和旧版设定不再作为新版正史。它们只能提供案件节奏、物件、女性站位、资源模块和升级爽点，不能反向修改新版正史边界。

# 新版正文主角核心视角与读感校准

## 文档定位

本文是正文审稿与改稿检查卡，不是新的设定边界。

本文只约束正文入口、视角锚点、环境描写和读感改稿手法，不改变章节收益、人物关系、时间线和卷末成果。若本文与 `新旧设定融合总则.md`、`墓碑镇历史爽文新设定.md`、`新版全书卷纲.md`、`第一卷章节细纲.md`、`新版正文前二十章合规复盘与防偏校准.md` 冲突，以上位文档为准。

## 核心判断

新版正文的网文感，首先来自主角核心视角。

主角伊莱亚斯·韦德是绝对核心。正文可以写环境、配角、历史人物和局势，但不能长时间脱离韦德的看见、判断、介入和夺取。

当前第一章暴露的问题不是单句太文，而是叙事入口偏外部镜头：先扫墓碑镇、艾伦街、银矿热、工资箱、尸体、矿工、牛仔、怀特和露丝，再让韦德入场。读者先读世界，再等主角，核心视角不清晰。

改稿目标：从“西部历史群像开场”改为“主角入场夺箱开场”。

## 开章视角规则

每章开头优先满足以下要求：

1. 尽早锚定韦德，最好前三段内出现韦德本人、韦德正在看的物件、或韦德即将介入的冲突。
2. 开头不要先做大范围环境扫描。地点、年代、街道、机构、配角身份，只保留服务当前冲突的最小信息。
3. 第一屏读者要知道：韦德面对什么麻烦，他要抢什么位置，谁挡着他。
4. 章节入口优先从动作、物件、冲突进入，而不是从背景设定进入。
5. 如果必须写配角或环境，必须很快回到韦德的视线、判断或行动上。

## 环境描写规则

环境描写不能脱离主角太久。

允许：

1. 韦德看见的街尘、枪套、工资票、箱锁、马蹄印、门牌、酒杯、印章。
2. 即将影响韦德判断或行动的环境细节。
3. 能帮助读者理解韦德将要夺取什么的物件。

避免：

1. 开头连续介绍城镇、街道、产业、建筑、路人和配角。
2. 先铺世界，再等主角出现。
3. 从全知镜头扫街，导致读者不知道该跟谁代入。
4. 环境细节没有进入韦德判断，也没有转化为冲突、证据或收益。

环境描写的最低要求：写完一个环境细节后，要能回答“这个细节如何帮助韦德判断、压场、夺权或制造爽点？”

## 配角与群像规则

配角可以先出现，但不能抢走叙事入口。

1. 怀特、露丝、克拉拉、艾达、贝汉、厄普、霍利迪等人物，都应围绕韦德的场面功能出场。
2. 配角身份不要在开头集中说明。优先用动作和资源表现，例如露丝先放下钥匙，克拉拉先护住铅字，艾达先按住工资票。
3. 配角单独行动可以写，但不能长时间脱离韦德主线。写完配角动作，要立刻回到韦德如何利用、回应、压住或反转。
4. 历史人物可以有强度，但视角不能追随他们太久。读者要始终知道他们是在试探韦德、挡韦德、见证韦德或让位给韦德。

## 第一章专项校准

第一章的读者记忆点必须是：韦德入场、工资箱不走、红胡子缴枪、矿工看见钱还在。

改稿方向：

1. 开头从工资箱正在被拖走、或韦德看见工资箱被拖走开始。
2. 墓碑镇、艾伦街、亚利桑那领地、银矿热等信息压缩到冲突中，不单独铺陈。
3. 怀特和露丝可以保留，但不要在韦德出场前占据太多段落。怀特代表镇上旧秩序压不住场，露丝代表旅店资源看见破绽，二者都应服务韦德入场。
4. 尸体、帽子、矿灯棉芯、马粪等细节可以保留，但要围绕韦德判断和工资箱冲突，不能成为独立画面展示。
5. 证据链不要写成现场报告。马蹄印、血流、湿泥、工资单要放进对抗里，让红胡子抢话、矿工逼近、怀特犹豫，韦德边压人边揭破。
6. 爽点落在红胡子半跪、枪被缴、工资箱不走、矿工敢往前一步。

## 场面检查句

审稿或改稿时逐场检查：

1. 这一场距离韦德太远了吗？
2. 读者是否知道韦德在哪里、看见什么、要做什么？
3. 环境描写是否服务韦德判断、压场或夺取？
4. 配角动作是否很快回到韦德的利用、回应、压制或反转？
5. 这场戏韦德当场夺了什么？
6. 敌人当场失去了什么？
7. 哪个物件换了主人、位置、用途或控制权？
8. 如果连续三段没有韦德、没有韦德要夺取的物件、没有韦德造成的站位变化，就要改。

## 负面清单

出现以下写法时，优先改：

1. 主角出场晚：开头连续铺环境、尸体、配角、路人，韦德迟迟不出现。
2. 外部镜头过重：像影视分镜一样扫街，读者无法进入韦德视角。
3. 世界先于主角：先讲墓碑镇如何、银矿热如何、街道如何，再写韦德做什么。
4. 配角先占场：怀特、露丝、贝汉、厄普等人在主角入场前获得过多解释。
5. 环境独立成段：风、尘、街道、建筑、尸体细节没有服务韦德行动。
6. 证据连环讲解：主角连续解释马印、血迹、账页、签名，敌人和旁人只听。
7. 纸面物件堆叠：县印、签名、名单、账本、铅字、证物箱连续出现，却没有明确归属变化。
8. 银星冷感无后果：纸边发冷、灯焰变低、残影出现后，没有公开证据或现实动作落地。

## 保留项

不要为了主角核心视角，把本书特色删薄：

1. 真实地名、人名、年代器物可以保留，但要嵌入韦德入场和冲突。
2. 枪、马、酒馆、县署、报纸、矿工、马车、牢门、钥匙、印章、工资箱等西部硬物件必须保留。
3. 银星眼冷感和诡异感可以保留，但必须落成韦德判断、公开证据或可见夺权。
4. 女性和历史人物可以有强度，但不能让视角长期离开韦德，更不能抢走韦德的定局权。

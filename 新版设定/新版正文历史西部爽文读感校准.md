# 新版正文历史西部爽文读感校准

## 文档定位

本文是正文审稿与改稿检查卡，不是新的设定边界。

本文只约束读感和改稿手法，不改变设定边界、章节收益、人物关系、时间线和卷末成果。若本文与 `新旧设定融合总则.md`、`墓碑镇历史爽文新设定.md`、`新版全书卷纲.md`、`第一卷章节细纲.md`、`新版正文前二十章合规复盘与防偏校准.md` 冲突，以上位文档为准。

## 读感目标

新版正文要保留历史西部质感，同时读起来像网文爽文：

1. 轻阅读：删冗余解释，不机械短句化，不削掉必要质感。
2. 强对抗：证据、文书、账本、名单不能只被讲解，必须被抢、挡、质疑、抵赖、签下、缴走或封住。
3. 可见夺权：每章都要让读者看见韦德夺了谁的权、保住谁的钱、缴了谁的枪、扣了谁的车、封了谁的门、逼谁退了一步。
4. 西部反馈：不追求普通玄幻式起哄打脸，不写满场叫好；用枪手手离枪、酒杯停住、矿工靠近、县里人不敢盖章、牌桌没人下注、女人把钥匙放到韦德手边等方式反馈爽点。

必须保留：

1. 真实地名、人名、年代器物和历史人物先强后让。
2. 枪、马、酒馆、县署、报纸、矿工、马车、牢门、钥匙、印章、工资箱等西部硬物件。
3. 银星眼冷感和诡异感，但冷感必须落成公开证据、现实后果或可见夺权。
4. 主角少话压场、女性以资源和站位介入、路人作为世界温度计。

## 改稿四优先

改稿时按以下顺序处理，不先大加新剧情：

1. 证据链对抗化：把主角连续讲解改成敌人抢话、抵赖、威胁、伸手抢物，韦德边压制边揭破。
2. 物件归属变化：枪、箱、钥匙、账本、名单、座位、马车、门、印章出现后，必须尽量产生位置、主人、用途或控制权变化。
3. 即时反馈：补敌人、路人、盟友、女性的动作反馈。反馈要贴合西部场面，不写泛起哄。
4. 少加新剧情：优先改句法、对话、站位、反馈和物件后果；只有章节承载不足时，才新增桥段并同步细纲。

## 场面检查句

审稿或改稿时逐场检查：

1. 这场戏韦德当场夺了什么？
2. 敌人当场失去了什么？
3. 哪个物件换了主人、位置、用途或控制权？
4. 路人、盟友、女性是否用动作反馈了结果？
5. 银星眼冷感是否落成公开证据或现实后果？
6. 如果连续三段只有解释、观察、气氛，没有人抢、挡、退、交、签、缴、封，就要改。

## 西部反馈清单

优先使用这类反馈替代普通起哄：

1. 枪手手离开枪套，或没敢把枪拔出来。
2. 酒杯停在半空，牌桌没人再下注。
3. 马车不再往前走，车夫不敢甩鞭。
4. 县里书记员不敢盖章，或者签名越写越小。
5. 矿工往前靠，帽子攥紧，工资票压到桌上。
6. 报童改喊新标题，报馆铅字重新排版。
7. 女人把钥匙、药瓶、账本、名单或房门位置放到韦德手边。
8. 敌人的椅子被空出来，门被封住，牢门钥匙去向被看见。

## 负面清单

出现以下写法时，优先改：

1. 证据连环讲解：主角连续解释马印、血迹、账页、签名，敌人和旁人只听。
2. 纸面物件堆叠：县印、签名、名单、账本、铅字、证物箱连续出现，却没有明确归属变化。
3. 银星冷感无后果：纸边发冷、灯焰变低、残影出现后，没有公开证据或现实动作落地。
4. 历史人物压过韦德：厄普、霍利迪、怀特、贝汉等人的判断或威望抢走主角定局权。
5. 女性只递材料：女性角色只提供纸、钥匙、名单，不用自己的职业、资源、站位制造后果。
6. 敌人太快认输：反派没有抢、挡、抵赖、威胁、转移、撕纸、拔枪或叫人，就直接签字低头。
7. 路人反馈太文明：底层矿工、枪手、车夫、报童在该急、该骂、该退、该靠近时过度安静。
8. 为了轻阅读乱删西部质感：把枪、马、酒馆、县署、报纸、矿工、铁路、马车等年代器物换成泛爽文场景。

## 样本警戒

### 第 001 章

读者记忆点应是工资箱、红胡子缴枪、韦德踩住拖箱绳、矿工看见钱还在。

改稿方向：

1. 开头不要让墓碑镇设定、银矿热、尸体、矿工、牛仔、露丝等信息压过工资箱冲突。
2. 证据链不要写成现场报告。马蹄印、血流、湿泥、工资单要放进对抗里，让红胡子抢话、矿工逼近、怀特犹豫，韦德边压人边揭破。
3. 爽点落在红胡子半跪、枪被缴、工资箱不走、矿工敢往前一步。

### 第 020 章

读者记忆点应是县里失去一个可见控制点，而不是证物很多。

改稿方向：

1. 县印铜模、私账、枪手名单、观察窄纸都可以保留，但每样物件都要尽量对应一个后果。
2. 巴特写承诺不能太顺，应有抵赖、拖延、威胁或旁人压力，再被韦德压下。
3. 县里势力必须失去可见控制点，例如备用链消息被锁死、马车路线被截、后桌被封、牌桌没人敢再收县印钱。
4. 怀亚特·厄普只作为外来强者观察对象，不进入脏活名单。
